## 1. 스크린 인벤토리

전체 사이트는 **12개 주요 화면**으로 구성됩니다.

| ID | 화면 | URL 패턴 | Phase | 우선순위 |
|----|------|---------|-------|---------|
| S01 | 홈페이지 | `/[locale]/` | 1 | Must |
| S02 | 플랫폼 목록 | `/[locale]/platforms/` | 1 | Must |
| S03 | 플랫폼 상세 | `/[locale]/platforms/[platform]/` | 1 | Must |
| S04 | 가이드 목록 | `/[locale]/guides/` | 1 | Must |
| S05 | 가이드 상세 | `/[locale]/guides/[slug]/` | 1 | Must (★핵심) |
| S06 | 활용 시나리오 | `/[locale]/use-cases/[slug]/` | 2 | Should |
| S07 | 비교 페이지 | `/[locale]/compare/[slug]/` | 2 | Should |
| S08 | 무료 한도 대시보드 | `/[locale]/tools/free-tier-dashboard/` | 2 | Should |
| S09 | 무료 한도 계산기 | `/[locale]/tools/free-tier-calculator/` | 2 | Should |
| S10 | 블로그 목록 | `/[locale]/blog/` | 2 | Could |
| S11 | 블로그 포스트 | `/[locale]/blog/[slug]/` | 2 | Could |
| S12 | 사용자 프로필 | `/[locale]/profile/` | 3 | Could |

**인터랙션 상세:**

| 요소 | 인터랙션 | 상세 |
|------|---------|------|
| 진행률 바 | sticky | 스크롤 시 상단 고정, 현재 단계 업데이트 (IntersectionObserver) |
| 목차 (데스크톱) | sticky + active | 현재 보고 있는 단계 하이라이트, 클릭 시 스무스 스크롤 |
| 목차 (모바일) | accordion | 접힌 상태 기본, 탭하여 펼침 |
| 스크린샷 | 하이라이트 오버레이 | 빨간 박스가 CSS position:absolute로 오버레이 |
| 스크린샷 (모바일) | pinch-zoom | 탭하면 라이트박스 모달로 확대 |
| 코드블록 | 복사 버튼 | 클릭 → 클립보드 복사 → "복사됨 ✓" 2초간 표시 |
| DevTip | 접힘/펼침 | 기본 접힌 상태, 클릭 시 slide-down 애니메이션 |
| Checkpoint | 분기 | "네" → 다음 단계 스크롤 / "안 됐어요" → 트러블슈팅 모달 |
| 이전/다음 버튼 | 네비게이션 | 해당 단계 ID로 스크롤 이동 |
| 피드백 👍/👎 | 제출 | 클릭 → API 호출 (또는 Google Forms) → "감사합니다" 표시 |

**상태 관리:**

  prerequisiteChecks: Record<string, boolean>;  // 선행 가이드 체크 상태
  feedbackSubmitted: boolean;    // 피드백 제출 여부
}
