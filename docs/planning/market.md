## 1. 시장 개요

### 1.1 전체 시장 규모

**클라우드 서비스 시장 (Global):**
- 2024년 글로벌 퍼블릭 클라우드 시장: 약 $6,790억 (Gartner 추정)
- 2025년 예상: 약 $8,000억+ (YoY 20%+ 성장)
- 2028년 전망: 약 $1.3조

**클라우드 교육/튜토리얼 시장:**
- 클라우드 교육 시장: 2024년 약 $50억, 2028년 $100억+ 전망
- 온라인 기술 교육 플랫폼(Udemy, Coursera, A Cloud Guru 등): 연간 $10억+ 시장
- 기술 블로그/튜토리얼 광고 시장: 정량화 어렵지만, DigitalOcean, Twilio 등이 튜토리얼 콘텐츠에 연간 수백만 달러 투자

### 1.2 우리가 노리는 세부 시장 (SAM → SOM)

```
TAM (전체 시장): 클라우드 서비스 사용자 전체
    → 약 수억 명의 개발자 + 비개발자

SAM (접근 가능 시장): 클라우드 무료 티어를 세팅하려는 사용자
    → "cloud free tier setup" 관련 검색량: 월 수십만 건 (영어 기준)
    → "GCP 무료" 관련 한국어 검색량: 월 수만 건

SOM (초기 타겟): 비개발자 + 주니어 개발자 중 한/영 검색 사용자
    → 월 10-50만 페이지뷰 규모
```

## 5. 시장 기회 분석

### 5.1 왜 이 시장에 빈틈이 있는가

1. **벤더 편향:** 공식 문서는 자사 제품만 다룹니다. "GCP와 AWS 중 뭐가 더 무료로 쓰기 좋은가?"에 대한 중립적 비교가 없습니다.

2. **개발자 편향:** 기존 튜토리얼은 "터미널에서 gcloud 명령어 치세요"부터 시작합니다. 콘솔(웹 UI)에서 클릭으로 세팅하는 가이드는 드뭅니다.

3. **파편화:** "GCP Vision API"를 검색하면 블로그 A, "AWS Lambda"를 검색하면 블로그 B가 나옵니다. 모든 무료 API를 한 곳에서 비교하고 세팅할 수 있는 원스톱 서비스가 없습니다.

4. **업데이트 부재:** 대부분의 블로그 포스트는 작성 후 업데이트되지 않습니다. 클라우드 콘솔 UI는 수시로 바뀌는데, 2년 전 스크린샷으로는 따라할 수 없습니다.

### 5.2 시장 진입 타이밍

```
[2020-2022]  클라우드 보편화 시작, 튜토리얼 수요 급증
[2023-2024]  AI 시대 개막, 비개발자의 API 관심 폭발
[2025 ★NOW] 무료 티어 경쟁 심화, 비개발자 타겟 가이드 부재
[2026-2027]  AI가 일부 가이드를 대체할 가능성, 그 전에 브랜드 확보 필요
```

**지금이 최적의 타이밍입니다.** AI가 스크린샷 기반 가이드까지 대체하려면 2-3년은 걸리고, 그 사이에 브랜드와 SEO 입지를 확보할 수 있습니다.

### 5.3 시장 성장 동인 (Growth Drivers)

1. **클라우드 네이티브 확산:** 더 많은 서비스가 클라우드 API 위에 구축됨
2. **AI API 대중화:** Gemini, GPT, Claude API를 일반인도 쓰고 싶어함
3. **노코드/로우코드 성장:** Zapier → API 직접 호출로의 자연스러운 진화
4. **스타트업 생태계 성장:** 초기 스타트업이 무료 티어로 MVP를 구축하려는 수요
5. **교육 시장 디지털화:** 학교/부트캠프에서 클라우드 실습 수요 증가

## 7. SWOT 분석

### Strengths (강점)
- 로보틱스 엔지니어로서 클라우드 API 실사용 경험
- 비개발자 관점 이해 (직접 세팅하며 겪은 어려움)
- 한/영 동시 콘텐츠 생산 가능
- 초기 비용 거의 0원 (Vercel + 무료 도구들)

### Weaknesses (약점)
- 1인 운영 → 콘텐츠 생산 속도 제한
- 스크린샷 업데이트 부담 (UI 변경 시)
- 초기 SEO 권위(Authority) 부재
- 마케팅/커뮤니티 운영 경험 제한

### Opportunities (기회)
- 비개발자 대상 클라우드 가이드 시장 공백
- AI API 관심 폭발적 증가
- 한국어 기술 콘텐츠 부족
- 오픈소스 기여 모델로 콘텐츠 확장 가능

### Threats (위협)
- AI(ChatGPT, Claude)가 가이드를 대체할 가능성
- 클라우드 벤더의 자체 가이드 강화
- SEO 알고리즘 변화
- 무료 티어 축소/폐지 가능성

## 8. 결론

### 핵심 시장 인사이트

1. **수요는 확실합니다.** 관련 키워드 월 검색량이 수십만 건이고, 증가 추세입니다.
2. **비개발자 타겟 시장에 명확한 공백이 있습니다.** 기존 경쟁자들은 모두 개발자 대상입니다.
3. **광고 수익성이 좋은 니치입니다.** 클라우드 관련 CPC가 높아 적은 트래픽으로도 의미 있는 수익이 가능합니다.
4. **타이밍이 맞습니다.** AI 시대에 비개발자의 API 관심이 급증하고 있고, AI가 스크린샷 가이드를 대체하기까지 2-3년의 윈도우가 있습니다.

**결론: 사이드 프로젝트로 시작하되 장기적 성장 가능성이 충분한 시장입니다.**
