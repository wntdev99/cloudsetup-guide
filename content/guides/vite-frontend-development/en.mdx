<Step number={4} title="Integrate with APIs">

### Step 4.1: Fetch Data with Vue

<CopyBlock
  code="<script setup>
import { ref, onMounted } from 'vue'

const data = ref(null)
const loading = ref(true)
const error = ref(null)

onMounted(async () => {
  try {
    const response = await fetch('https://api.example.com/data')
    data.value = await response.json()
  } catch (e) {
    error.value = e.message
  } finally {
    loading.value = false
  }
})
</script>

<template>
  <div v-if='loading'>Loading...</div>
  <div v-else-if='error'>Error: {  { error } }</div>
  <div v-else>{  { data } }</div>
</template>"
  language="vue"
/>

### Step 4.2: React API Integration

<CopyBlock
  code="import { useState, useEffect } from 'react'

export function DataComponent() {
  const [data, setData] = useState(null)
  const [loading, setLoading] = useState(true)
  const [error, setError] = useState(null)

  useEffect(() => {
    fetch('https://api.example.com/data')
      .then(res => res.json())
      .then(data => setData(data))
      .catch(err => setError(err))
      .finally(() => setLoading(false))
  }, [])

  if (loading) return <div>Loading...</div>
  if (error) return <div>Error: {error}</div>
  return <div>{JSON.stringify(data)}</div>
}"
  language="javascript"
/>

## Optimization Tips

<CopyBlock
  code="Performance Optimizations:

1. Code Splitting
   - Dynamic imports reduce initial bundle
   - Lazy load routes and components

2. Image Optimization
   - Use WebP format
   - Optimize dimensions
   - Compress before upload

3. Caching
   - Use ETags and cache headers
   - Set proper expires headers
   - Use service workers

4. Bundling
   - Remove unused code (tree-shaking)
   - Minify CSS and JS
   - Use gzip compression"
  language="text"
/>
