<Step number={1} title="Sign Up for Sentry">

### Visit Sentry Website

1. Go to [Sentry.io](https://sentry.io)

1. Enter your email address
2. Create a password (minimum 8 characters)

<Step number={2} title="Verify Email and Set Up Organization">

### Click Verification Email

1. Check your email inbox
2. Open the **verification email** from Sentry
3. Click the **Verify Email** link

### Enter Organization Information

Sentry manages projects within **organizations**:

1. Enter organization name (e.g., "My Company", "Personal Projects")
2. Organization slug [URL] is auto-generated
3. Click **Create Organization**

<Step number={3} title="Create Your First Project">

### Project Creation Page

After organization creation, you're redirected to **Create Project** page.

1. Click **Create Project** button
2. Select your platform:
   - JavaScript / React / Vue
   - Python / Django / Flask
   - Node.js / Express
   - Java / Spring
   - Go
   - Ruby
   - .NET
   - PHP
   - Other platforms

### Configure Project

In the project creation dialog:

1. Enter **project name** (e.g., "My Web App", "API Server")
2. Select **platform** (your programming language)
3. Click **Create Project**

<Step number={4} title="Configure Project and Get DSN">

### Find DSN (Data Source Name)

After project creation, find DSN in the **Settings** page.

DSN looks like:
```
https://key@sentry.io/projectid
```

### Locate DSN

1. Click **Settings** in left navigation
2. Select **Client Keys**
3. DSN is displayed

### Review Settings

In **Project Settings** page:

1. **General** - Project name, team info
2. **Error Reporting** - Error filter settings
3. **Performance** - Enable performance monitoring
4. **Releases** - Version management

## Tips and Best Practices

### Invite Team Members

1. Go to **Settings** > **Team**
2. Click **Invite Team Members**
3. Enter team member email
4. Select role:
   - Owner - Full permissions
   - Manager - Project management
   - Member - Read-only

### Separate Environments

Create separate projects for different environments:
- **Production** - Live environment
- **Staging** - Pre-production
- **Development** - Development environment

### Set Up Alerts

1. Access **Alerts** menu
2. Create **Alert Rules**
3. Set notifications by severity or frequency
4. Receive notifications via Slack, email, etc.

## Troubleshooting

### Verification Email Didn't Arrive

1. Check **Spam folder**
2. Use **[Sentry password reset](https://sentry.io/account/password/reset/)**
3. Try signing up with different email

### Can't See Projects

1. Verify correct **organization** is selected
2. Check **Projects** menu in left navigation
3. Refresh the page

### Already Have Sentry Account

If you already have an account, sign in at [Sentry.io](https://sentry.io) and choose **Create Project**.
