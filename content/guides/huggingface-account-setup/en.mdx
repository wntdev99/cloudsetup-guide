## Creating Your API Token

For programmatic access to models:

1. Click your avatar ‚Üí **Settings**
2. Go to **Access Tokens**
3. Click **New token**
4. Give it a name: "My API Token"
5. Select **Read** access level
6. Click **Generate token**
7. Copy and save the token (shown only once!)

<Callout type="warning">
‚ö†Ô∏è **Protect your API token**

- Never commit to GitHub
- Store in .env file
- Treat like a password
- Can be regenerated if compromised
</Callout>

## Frequently Asked Questions

<Callout type="info">
**Q. Is Hugging Face really free?**

A. Yes! Core features are completely free. Premium services like commercial model hosting and priority inference are paid.
</Callout>

<Callout type="info">
**Q. Can I use Hugging Face models commercially?**

A. Depends on the model's license. Most models have MIT or Apache 2.0 licenses allowing commercial use. Check each model's card for license info.
</Callout>

<Callout type="info">
**Q. How do I download models locally?**

A. Use the Hugging Face `transformers` library:

```python
from transformers import AutoModel
model = AutoModel.from_pretrained("model-name")
```

This downloads the model to your machine for offline use.
</Callout>

<Callout type="info">
**Q. What's the difference between models and datasets?**

A. Models are the AI weights trained on data. Datasets are collections of data used to train or evaluate models. Hugging Face hosts both.
</Callout>

## Troubleshooting

### Verification email not received

- Check spam/junk folder
- Wait 5 minutes for email delivery
- Try resending from the signup page
- Use GitHub/Google login if email fails

### Username already taken

- Usernames are globally unique
- Try adding numbers: `username123`
- Try adding your location: `username-us`
- Cannot change username after creation

### Can't access models

- Check if model is private
- Some models require authentication
- Verify your account is confirmed
- Try browser's incognito mode

### Can't download models

- Ensure stable internet connection
- Check disk space available
- Try `huggingface-hub` library instead of manual download
- Some large models may timeout - be patient
