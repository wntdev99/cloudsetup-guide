# Hugging Face Hub 계정 만들기 - 첫 번째 AI 모델 시작

Hugging Face Hub는 전 세계에서 가장 큰 오픈 소스 AI 모델 저장소입니다. 언어 모델(LLM)부터 컴퓨터 비전 모델까지 수천 개의 모델을 무료로 사용할 수 있습니다. 지금 바로 계정을 만들고 10분 안에 시작해보세요!

## Hugging Face란 무엇인가요?

<Callout type="info">
🤗 **Hugging Face는 AI 모델의 집**

- **모델 허브**: 50만 개 이상의 모델과 데이터셋 접근
- **커뮤니티 중심**: 전 세계의 연구자와 개발자가 모델 공유
- **무료 추론**: 허브에서 바로 모델 테스트 가능
- **통합 완벽**: PyTorch, TensorFlow, JAX와 호환
- **데이터셋**: 5만 개 이상의 학습/평가 데이터셋

이렇게 사용됩니다:
- 머신러닝 연구자
- AI 애플리케이션 개발자
- AI를 배우는 학생
- 프로덕션 배포
</Callout>

### 방법 A: 이메일로 가입

1. **Continue with Email**을 클릭하세요
2. 이메일 주소를 입력하세요
3. 안전한 비밀번호를 설정하세요 (최소 8자)
4. **I agree to Hugging Face** 체크박스를 확인하세요
5. **Create Account**를 클릭하세요

### 방법 B: 소셜 로그인

1. **Continue with GitHub** 또는 **Continue with Google**을 클릭하세요

3. 3단계로 넘어가세요

## 프로필 설정하기

가입 후 Hugging Face 경험을 개인화하세요:

### 1. 프로필 사진 추가
- 아바타 클릭 (오른쪽 상단)
- **Settings** 이동
- 프로필 이미지 업로드
- 모델 리포와 댓글에 표시됨

### 2. 소개 설정
- **Settings** → **Account** 이동
- 소개 추가 (예: "ML Engineer | Python | AI enthusiast")
- 포트폴리오나 GitHub 링크 추가

### 3. 알림 활성화
- **Notifications** 설정으로 이동
- 즐겨찾는 모델 업데이트 받기
- 토론 알림 받기

### 4. GitHub 연결
- **Settings** → **Social accounts**
- GitHub 연결하여 오픈 소스 작업 표시
- 프로젝트 교차 홍보

## API 토큰 생성하기

프로그래밍 방식으로 모델에 접근하려면:

1. 아바타 클릭 → **Settings**
2. **Access Tokens** 이동
3. **New token** 클릭
4. 이름 지정: "My API Token"
5. **Read** 접근 수준 선택
6. **Generate token** 클릭
7. 토큰 복사 및 저장 (한 번만 표시됨!)

<Callout type="warning">
⚠️ **API 토큰 보호하기**

- GitHub에 커밋하지 마세요
- .env 파일에 저장하세요
- 비밀번호처럼 취급하세요
- 손상된 경우 재생성 가능
</Callout>

## 자주 묻는 질문

<Callout type="info">
**Q. Hugging Face가 정말 무료인가요?**

A. 네! 핵심 기능은 완전히 무료입니다. 상용 모델 호스팅 및 우선 추론과 같은 프리미엄 서비스는 유료입니다.
</Callout>

<Callout type="info">
**Q. Hugging Face 모델을 상용으로 사용할 수 있나요?**

A. 모델의 라이선스에 따라 다릅니다. 대부분의 모델은 상용 사용을 허용하는 MIT 또는 Apache 2.0 라이선스를 가지고 있습니다. 각 모델 카드에서 라이선스 정보를 확인하세요.
</Callout>

<Callout type="info">
**Q. Hugging Face 모델을 로컬로 다운로드하려면?**

A. Hugging Face 변환기 라이브러리를 사용하세요:

from transformers import AutoModel
model = AutoModel.from_pretrained("model-name")

이는 모델을 오프라인 사용을 위해 컴퓨터에 다운로드합니다.
</Callout>

<Callout type="info">
**Q. 모델과 데이터셋의 차이는?**

A. 모델은 데이터로 훈련된 AI 가중치입니다. 데이터셋은 모델 훈련 또는 평가에 사용되는 데이터 모음입니다. Hugging Face는 둘 다 호스팅합니다.
</Callout>

## 문제 해결

### 확인 이메일을 받지 못했습니다

- 스팸/정크 폴더 확인
- 이메일 배송 5분 대기
- 가입 페이지에서 재전송 시도
- 이메일이 실패하면 GitHub/Google 로그인 사용

### 사용자명이 이미 사용 중입니다

- 사용자명은 전 세계적으로 고유합니다
- 숫자 추가 시도: `username123`
- 위치 추가 시도: `username-us`
- 생성 후 사용자명 변경 불가

### 모델에 접근할 수 없습니다

- 모델이 비공개인지 확인
- 일부 모델은 인증 필요
- 계정이 확인되었는지 확인
- 브라우저 시크릿 모드 시도

### 모델을 다운로드할 수 없습니다

- 안정적인 인터넷 연결 확인
- 사용 가능한 디스크 공간 확인
- 수동 다운로드 대신 변환기 라이브러리 시도
- 일부 대형 모델은 타임아웃 가능 - 기다려주세요
