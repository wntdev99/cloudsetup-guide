# Vite 설치 - 빠른 프론트엔드 빌드 도구

Vite는 현대적인 웹 개발을 위한 초고속 빌드 도구입니다. 15분 안에 첫 번째 Vite 프로젝트를 만들고 즉각적인 HMR(핫 모듈 교체)을 경험하세요!

## Vite란?

<Callout type="info">
⚡ **Vite는 빠른 프론트엔드 개발입니다**

- **즉시 서버 시작**: 1초 미만의 시작 시간
- **초고속 HMR**: 변경 사항이 즉시 반영됨
- **최적화된 빌드**: 프로덕션을 위한 스마트 번들링
- **프레임워크 무관**: Vue, React, Svelte와 작동
- **네이티브 ESM**: 개발에서 ES 모듈 사용
- **개발 및 프로덕션**: 동일한 도구, 다르게 최적화됨

이렇게 사용됩니다:
- 단일 페이지 애플리케이션 (SPA)
- 컴포넌트 개발
- 웹 게임
- 프로그레시브 웹 앱
- 빠른 프로토타이핑
</Callout>

<Step number={1} title="첫 번째 Vite 프로젝트 만들기">

### Step 1.1: npm으로 프로젝트 만들기

터미널을 열고 실행하세요:

<CopyBlock

또는 yarn 사용:

/>

### Step 1.2: 템플릿 선택

<CopyBlock
  code="사용 가능한 템플릿:

vanilla-ts       TypeScript
vue              Vue 3

react            React with JSX
react-ts         React + TypeScript
react-swc        React with SWC compiler
preact           Preact
preact-ts        Preact + TypeScript
lit              Lit
lit-ts           Lit + TypeScript
svelte           Svelte
svelte-ts        Svelte + TypeScript"

### Step 1.3: 이동 및 설치

## Vite 구성

`vite.config.js` 만들기:

<CopyBlock
  code="import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  server: {
    port: 3000,
    open: true,
    host: 'localhost'
  },
  build: {
    outDir: 'dist',
    sourcemap: false
  }
})"
  language="javascript"
/>

## 파일 구조 설명

<CopyBlock
  code="중요한 파일:

index.html
  - 진입점
  - Vite가 여기서 모듈 발견

src/main.js (또는 .ts)
  - 애플리케이션 진입점
  - 스타일, 컴포넌트 임포트

vite.config.js
  - Vite 구성
  - 플러그인, 최적화 설정

package.json
  - 프로젝트 메타데이터
  - 종속성
  - 스크립트

dist/
  - 프로덕션 빌드 출력
  - 배포할 파일"
  language="text"
/>
