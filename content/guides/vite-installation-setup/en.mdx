Vite is a lightning-fast build tool for modern web development. Create your first Vite project in 15 minutes and experience instant HMR (Hot Module Replacement)!

## What is Vite?

<Callout type="info">
âš¡ **Vite is fast frontend development**

- **Instant Server Start**: Sub-second startup time
- **Lightning-Fast HMR**: Changes reflect instantly
- **Optimized Builds**: Smart bundling for production
- **Framework Agnostic**: Works with Vue, React, Svelte

<Step number={2} title="Start Development Server">

### Step 2.1: Run Dev Server

<CopyBlock
  code="npm run dev"
  language="bash"
/>

You'll see:

<CopyBlock
  code="VITE v5.0.0  ready in 245 ms

  Local:    http://localhost:5173/
  press h to show help"
  language="bash"
/>

Open [http://localhost:5173](http://localhost:5173) in browser.

### Step 2.2: Try Hot Module Replacement

1. Open `src/main.js`
2. Change something:

<CopyBlock
  code="// src/main.js
document.querySelector('#app').innerHTML = 'Hello Vite!';"
  language="javascript"
/>

3. Save file - browser updates instantly!

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/vite-installation-setup-step2-en.png"
  alt="Dev server running"
  caption="Development server running with HMR enabled, instant updates"
/>

<DevTip>
**HMR Benefits:**

- No page reload needed
- State preserved between updates
- Only changed modules re-evaluated
- Instant feedback while developing
</DevTip>

</Step>

## Vite Configuration

Create `vite.config.js`:

<CopyBlock
  code="import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  server: {
    port: 3000,
    open: true,
    host: 'localhost'
  },
  build: {
    outDir: 'dist',
    sourcemap: false
  }
})"
  language="javascript"
/>
