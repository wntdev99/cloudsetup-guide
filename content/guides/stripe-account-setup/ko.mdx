# Stripe 계정 만들기 - 결제 처리 설정

Stripe는 **온라인 결제를 받는 가장 쉬운 방법**입니다. 이 가이드를 따라하면 20분 안에 계정을 만들고 기본 설정을 완료할 수 있습니다. 실제 돈을 받을 때만 수수료를 내면 됩니다.

## 시작하기 전에

</Callout>

<Step number={1} title="Stripe에 접속하여 가입 시작">

1. 웹브라우저에서 [Stripe.com](https://stripe.com)에 접속합니다.
2. 우측 상단의 **가입** 버튼을 클릭합니다.
3. 이메일 주소를 입력합니다.
4. 비밀번호를 만듭니다 (최소 8자, 숫자와 기호 포함).
5. **계속** 버튼을 클릭합니다.

  caption="이메일을 입력하고 강력한 비밀번호를 만드세요"
  priority={true}
/>

<Callout type="tip">
💡 **팁**: 회사 이메일을 사용하면 나중에 팀 멤버 권한 관리가 더 쉬워집니다.
</Callout>

</Step>

---

<Step number={2} title="이메일 확인">

1. 이메일 받은편지함에서 Stripe 확인 메시지를 확인합니다.
2. 이메일의 **이메일 확인** 링크를 클릭합니다.
3. Stripe 웹사이트로 돌아오면 자동으로 로그인됩니다.

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/stripe/step2-verify-email-ko.png"
  alt="이메일 확인 화면"
  caption="Stripe가 확인 링크를 이메일로 보냅니다"
/>

<Callout type="warning">
⚠️ **주의**: 확인 링크는 24시간 후 만료됩니다. 만료된 경우 새로운 확인 이메일을 요청하세요.
</Callout>

</Step>

---

<Step number={3} title="사업 정보 작성">

1. **사업명**을 입력합니다 (프리랜서인 경우 이름).
2. **사업 유형**을 선택합니다:
   - 개인/자영업자
   - 파트너십
   - 법인
   - 비영리단체
3. **사업 주소**를 입력합니다.
4. **전화번호**를 입력합니다.
5. **사업 설명**을 입력합니다 (예: "수제 쥬얼리 온라인 판매").
6. **계속** 버튼을 클릭합니다.

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/stripe/step3-business-profile-ko.png"
  alt="사업 정보 입력 양식"
  caption="정확한 사업 정보를 입력하여 검증을 통과하세요"
/>

<DevTip>
프리랜서나 부업인 경우 "개인/자영업자"를 선택하세요. 이것이 가장 빠른 승인 과정입니다.
</DevTip>

</Step>

---

<Step number={4} title="은행 정보 추가">

1. **국가**를 선택합니다 (은행 요구사항 설정).
2. **계좌 번호**와 **계좌 라우팅 번호**를 입력합니다.
3. **계좌 소유자 명**을 입력합니다 (은행 기록과 일치해야 함).
4. **계속** 버튼을 클릭합니다.

<Step number={5} title="전화번호 확인 및 대시보드 접근">

2. **전화번호 확인** 필드에 코드를 입력합니다.
3. **확인** 버튼을 클릭합니다.
4. 이제 **Stripe 대시보드**에 접근할 수 있습니다.

## 자주 묻는 질문

<Callout type="info">
**Q. 계정 승인에 얼마나 걸리나요?**

A. 대부분의 계정은 즉시 승인됩니다. 일부는 추가 검증이 필요할 수 있습니다 (1-3 영업일). 특히 고위험 업종이나 대량 거래의 경우입니다.
</Callout>

<Callout type="info">
**Q. 계정 생성에 수수료가 있나요?**

A. 아니요. Stripe 계정을 만들고 유지하는 것은 완전히 무료입니다. 성공한 결제만 처리하면 수수료를 냅니다.
</Callout>

<Callout type="info">
**Q. 테스트 모드는 무엇인가요?**

A. 테스트 모드는 실제 돈 없이 결제를 시뮬레이션합니다. 테스트 카드 4242 4242 4242 4242를 사용하여 테스트 거래를 할 수 있습니다. 실제 결제를 하려면 라이브 모드로 전환하세요.
</Callout>

<Callout type="info">
**Q. Stripe는 어떤 결제 수단을 지원하나요?**

A. Stripe는 신용카드 (Visa, Mastercard, Amex), 디지털 지갑 (Apple Pay, Google Pay), 계좌이체 등을 국가에 따라 지원합니다.
</Callout>

## 문제 해결

### 휴대폰 인증 코드가 작동하지 않습니다

- 시간 제한 내에 코드를 입력하고 있는지 확인하세요 (보통 10분)
- 코드에 여분의 공백이 없는지 확인하세요
- 만료된 경우 새 코드를 요청하세요

### 은행 검증에 시간이 오래 걸립니다

- Stripe가 계좌에 작은 테스트 입금 2개를 만듭니다 (1-2 영업일)
- 입금이 나타나면 Stripe 대시보드에서 금액을 확인해야 합니다
- 2주 내에 입금이 도착하지 않으면 Stripe 지원팀에 문의하세요

### 계정이 제한된 기능으로 표시됩니다

- Stripe가 추가 검증 문서를 요청할 수 있습니다
- Stripe 대시보드에서 요청 사항을 확인하세요
- 대기 중에도 테스트 결제는 계속 처리할 수 있습니다
