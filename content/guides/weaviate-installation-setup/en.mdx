# Install and Run Weaviate Vector Database

**Weaviate** is an open-source vector database that stores and searches vector embeddings. You can store vectors generated from machine learning models and perform semantic search. Follow this guide to install and run Weaviate using Docker.

<Step number={2} title="Run Weaviate Docker Image">

### Simple Docker Run

Run Weaviate the simplest way.

<CopyBlock language="bash" title="Start Weaviate container">
docker run -d \
  -p 8080:8080 \
  -p 50051:50051 \
  --name weaviate \
  semitechnologies/weaviate:latest
</CopyBlock>

### Use Docker Compose (Recommended)

For easier management, create a docker-compose.yml file.

<CopyBlock language="bash" title="Create docker-compose.yml file">
cat > docker-compose.yml << EOF
version: '3.4'
services:
  weaviate:
    image: semitechnologies/weaviate:latest
    ports:
      - "8080:8080"
      - "50051:50051"
    environment:
      QUERY_DEFAULTS_LIMIT: 25
      AUTHENTICATION_ANONYMOUS_ACCESS_ENABLED: 'true'
      PERSISTENCE_DATA_PATH: '/var/lib/weaviate'
    volumes:
      - weaviate_data:/var/lib/weaviate

volumes:
  weaviate_data:
EOF
</CopyBlock>

### Start with Docker Compose

<CopyBlock language="bash" title="Run Docker Compose">
docker-compose up -d
</CopyBlock>

<Step number={4} title="Install Python Client and Connect">

### Install Python Client Library

</CopyBlock>

### Connect in Python

<CopyBlock language="python" title="Python connection script">
import weaviate

client = weaviate.Client('http://localhost:8080')

print(client.is_ready())
</CopyBlock>

### Install Node.js Client

<CopyBlock language="bash" title="Install Node.js client">
npm install weaviate-client
</CopyBlock>

### Connect in Node.js

<CopyBlock language="javascript" title="Node.js connection script">
const weaviate = require('weaviate-client');

const client = weaviate.client({
  scheme: 'http',
  host: 'localhost:8080',
});

client.misc.readyChecker()
  .then(ready => console.log(ready ? 'Ready!' : 'Not ready'))
  .catch(err => console.error('Error:', err));
</CopyBlock>

## Weaviate Basics

### Vector Embeddings

Vector embeddings convert text, images, etc. to numeric arrays:

<CopyBlock language="text" title="Embedding Example">
Text: "dog" -> [0.234, 0.546, 0.123, ...]
Text: "puppy" -> [0.245, 0.567, 0.134, ...]
</CopyBlock>

Similar meanings have similar vectors.

### Schema

Define data structure (schema) in Weaviate first:

<CopyBlock language="python" title="Define schema">
class_obj = {
  'class': 'Article',
  'description': 'A blog post article',
  'properties': [
    {
      'name': 'title',
      'dataType': ['text']
    },
    {
      'name': 'content',
      'dataType': ['text']
    }
  ]
}
</CopyBlock>

## Troubleshooting

### Port Conflict

If port 8080 is in use:

<CopyBlock language="bash" title="Use different port">
docker run -d -p 9000:8080 --name weaviate semitechnologies/weaviate:latest
</CopyBlock>

Then access at `http://localhost:9000`.

### Out of Memory Error

Increase Docker memory allocation:

In Docker Desktop Settings > Resources, set memory to 4GB or more.

### Container fails to start

Check logs:

<CopyBlock language="bash" title="Check logs">
docker logs weaviate
</CopyBlock>

### Client connection fails

1. Verify Weaviate is running
2. Confirm URL is correct
3. Check firewall/network settings
