<Step number={2} title="Set App Permissions">

Configure what permissions your app will have in the workspace.

### Set Bot Token Scopes

Click **OAuth & Permissions** in left menu:

1. **Bot Token Scopes** section
2. Click **Add an OAuth Scope** button
3. Add required permissions:

Popular permissions:
- `chat:write`: Send messages
- `chat:read`: Read messages
- `users:read`: Get user info
- `app_mentions:read`: Detect bot mentions
- `messages.im`: Direct messages

<Step number={3} title="Copy Bot Token and API Key">

### Generate Bot Token

1. In OAuth & Permissions page
2. Click **Install to Workspace** button
3. Review permissions and click **Allow**

### Copy Token

1. Find **Bot User OAuth Token**
2. Copy token value [starts with xoxb-]

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/slack/slack-app-creation-step3-en.png"
  alt="Copy token"
  caption="Copy Bot User OAuth Token"
/>

### Securely Store Token

Save in .env file:

<CopyBlock
  code="SLACK_BOT_TOKEN=xoxb-your-token
SLACK_SIGNING_SECRET=your-signing-secret"
  language="bash"
/>

Add to .gitignore:

‚ö†Ô∏è **Token Security Precautions**

- Never share token publicly

- Regenerate immediately if exposed

<Step number={4} title="Setup Event Subscriptions">

Configure your app to listen for specific events.

### Access Events Menu

Click **Event Subscriptions** in left menu

### Set Request URL

1. Enter **Request URL** field
2. Specify URL where Slack sends events
3. Example: `https://your-domain.com/slack/events`

<Callout type="info">
**Why Request URL is needed**

- Slack POSTs events to this URL
- Your app receives and processes them
- Must be public URL [localhost not allowed]
</Callout>

### Add Event Subscriptions

1. **Subscribe to bot events** section
2. Click **Add Bot User Event**
3. Select events:

Popular events:
- `message.channels`: Channel messages
- `app_mention`: Bot mentions
- `reaction_added`: Reaction added
- `user_change`: User info changed

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/slack/slack-app-creation-step4-en.png"
  alt="Event subscriptions"
  caption="Configure Event Subscriptions"
/>

### Register Slash Commands [Optional]

In **Slash Commands** menu:

1. Click **Create New Command**
2. Enter command: `/mycommand`
3. Request URL: `https://your-domain.com/slack/commands`
4. Enter description
5. Click **Save**

## Copy Signing Secret

Signing Secret is needed to verify requests are from Slack.

1. **Basic Information** menu
2. **App Credentials** section
3. Copy **Signing Secret**

Add to .env file:

<CopyBlock
  code="SLACK_SIGNING_SECRET=your-signing-secret"
