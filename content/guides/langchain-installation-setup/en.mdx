# Install LangChain - Build LLM Applications in Python

LangChain is a powerful framework for building applications with large language models (LLMs). It simplifies working with models like GPT-4, Claude, and open-source alternatives. Let's set it up in 15 minutes!

## What is LangChain?

<Callout type="info">
üîó **LangChain simplifies LLM development**

- **Model Integration**: Connect to OpenAI, Hugging Face, Anthropic, Cohere
- **Chains**: Combine multiple LLM calls into workflows
- **Memory**: Add conversation history and context
- **Tools**: Integrate APIs, databases, search engines
- **Agents**: Create autonomous systems that make decisions
- **RAG**: Retrieve and augment information with documents

Perfect for:
- Chatbots and conversational AI
- Document analysis and summarization
- Question answering systems
- Automated workflows
- Content generation
</Callout>

### Step 2.2: Get Your API Key

For OpenAI (most popular):

1. Go to [OpenAI API Keys](https://platform.openai.com/api-keys)
2. Sign up or log in to your OpenAI account
3. Click **Create new secret key**
4. Copy the key (it won't show again!)
5. Save to environment variable:

<CopyBlock
  code="# On macOS/Linux
export OPENAI_API_KEY='sk-xxxxxxxxxxxx'

# On Windows (Command Prompt)
set OPENAI_API_KEY=sk-xxxxxxxxxxxx

# Or add to .env file
OPENAI_API_KEY=sk-xxxxxxxxxxxx"
  language="bash"
/>

### Step 4.2: Your First Chain

<CopyBlock
  code="from langchain_openai import ChatOpenAI
from langchain.prompts import ChatPromptTemplate
from langchain.chains import LLMChain

llm = ChatOpenAI(model='gpt-3.5-turbo')

# Define a prompt template
prompt = ChatPromptTemplate.from_template(
  'Tell me about {topic} in 2 sentences.'
)

# Create chain
chain = LLMChain(llm=llm, prompt=prompt)

# Run chain
result = chain.invoke({'topic': 'machine learning'})
print(result['text'])"
  language="python"
/>

## Common Installation Issues

### ImportError: No module named 'langchain'

Make sure LangChain is installed:

<CopyBlock
  code="pip install langchain --upgrade"
  language="bash"
/>

### API Key Not Found

Check your environment setup:

<CopyBlock
  code="# Verify API key is set
python -c 'import os; print(os.getenv(\"OPENAI_API_KEY\"))'

# If None, set it:
export OPENAI_API_KEY=sk-your-key-here"
  language="bash"
/>

### SSL Certificate Error

Update certificates on macOS:

<CopyBlock
  code="/Applications/Python 3.x/Install Certificates.command"
  language="bash"
/>
