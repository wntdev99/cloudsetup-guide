<Step number={1} title="Install with Docker (Recommended)">

Docker makes it quick and easy to get started.

### Check Docker Installation

Open a terminal and run:

<CopyBlock language="bash" title="Check Docker version">
docker --version
</CopyBlock>

If Docker isn't installed, download it from the [Docker official website](https://www.docker.com/products/docker-desktop).

### Run Redis with Docker

<CopyBlock language="bash" title="Start Redis Docker container">
docker run -d \
  --name redis \
  -p 6379:6379 \
  redis:7-alpine
</CopyBlock>

Command explanation:
- `-d`: Run in background
- `--name redis`: Container name
- `-p 6379:6379`: Port mapping
- `redis:7-alpine`: Lightweight Redis image

<Step number={3} title="Manual Installation - Linux (Ubuntu/Debian)">

### Install Package

<CopyBlock language="bash" title="Install Redis package">
sudo apt-get update
sudo apt-get install redis-server
</CopyBlock>

### Start Redis Server

<CopyBlock language="bash" title="Start service">
sudo service redis-server start
</CopyBlock>

### Enable Auto-start on Boot

<CopyBlock language="bash" title="Enable auto-start">
sudo update-rc.d redis-server defaults
</CopyBlock>

<Step number={4} title="Verify Redis Connection">

### Test Connection with redis-cli

<CopyBlock language="bash" title="Test with redis-cli">
redis-cli ping
</CopyBlock>

Response:
```
PONG
```

If you see `PONG`, Redis is working correctly.

### Connect in Interactive Mode

<CopyBlock language="bash" title="Interactive mode">
redis-cli
</CopyBlock>

Then enter commands:

<CopyBlock language="bash" title="Redis command examples">
SET mykey "Hello Redis"
GET mykey
</CopyBlock>

Response:
```
OK
"Hello Redis"
```

<Step number={5} title="Learn Basic Redis Commands">

Learn the most common Redis commands.

### Store and Retrieve Strings

<CopyBlock language="bash" title="String operations">
SET username "john"
GET username
</CopyBlock>

Response:
```
OK
"john"
```

### Set Expiration Time (TTL)

Automatically delete after 10 seconds:

<CopyBlock language="bash" title="Set TTL">
SET session "abc123" EX 10
GET session
</CopyBlock>

After 10 seconds:
```
(nil)
```

### List Operations

<CopyBlock language="bash" title="List operations">
LPUSH messages "Hello"
LPUSH messages "World"
LRANGE messages 0 -1
</CopyBlock>

Response:
```
(integer) 1
(integer) 2
1) "World"
2) "Hello"
```

### Hash Operations

<CopyBlock language="bash" title="Hash operations">
HSET user:1 name "John" age 30
HGETALL user:1
</CopyBlock>

Response:
```
(integer) 2
1) "name"
2) "John"
3) "age"
4) "30"
```

### Set Operations

<CopyBlock language="bash" title="Set operations">
SADD tags "python" "redis" "database"
SMEMBERS tags
</CopyBlock>

Response:
```
(integer) 3
1) "python"
2) "redis"
3) "database"
```

## Troubleshooting

### Port 6379 Already in Use

Run on a different port:

<CopyBlock language="bash" title="Use different port">
docker run -d \
  --name redis \
  -p 6380:6379 \
  redis:7-alpine
</CopyBlock>

Then connect to the different port:

<CopyBlock language="bash" title="Connect to different port">
redis-cli -p 6380 ping
</CopyBlock>

### redis-cli Not Found

If using Docker:

<CopyBlock language="bash" title="Run redis-cli in container">
docker exec -it redis redis-cli
</CopyBlock>

### Redis Server Not Responding

1. Check if server is running:
   ```bash
   docker ps | grep redis
   ```

2. Check logs:
   ```bash
   docker logs redis
   ```

3. Check port:
   ```bash
   netstat -an | grep 6379
   ```
