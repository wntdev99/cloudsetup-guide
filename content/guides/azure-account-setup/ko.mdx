# Azure 계정 만들기 - $200 무료 크레딧 받기

Microsoft Azure는 **글로벌 클라우드 플랫폼**으로, 가상 머신, 데이터베이스, 웹앱, AI 등 1000개 이상의 서비스를 제공합니다. 새로운 사용자는 **$200의 무료 크레딧**과 **12개월 무료 서비스**로 시작할 수 있습니다. 이 가이드를 따라하면 15분 안에 Azure 계정을 만들 수 있습니다.

<Step number={1} title="Azure 홈페이지 접속">

Microsoft Azure 가입 페이지로 이동합니다.

1. 웹브라우저에서 [Azure 시작 페이지](https://azure.microsoft.com)에 접속합니다.
2. 우측 상단의 **"무료 시작"** 또는 **"계정 만들기"** 버튼 클릭
3. Microsoft 계정으로 로그인하거나 새로운 계정 생성

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/azure/azure-account-setup-step1-ko.png"
  alt="Azure 홈페이지"
  caption="Azure 홈페이지에서 무료 시작 버튼을 클릭합니다"
/>

<Callout type="tip">
💡 **팁**: Microsoft 계정이 없다면 회사 이메일, Gmail, Yahoo 메일 등 어떤 이메일이든 사용할 수 있습니다.
</Callout>

</Step>

---

<Step number={2} title="개인 정보 입력">

Azure 계정 생성을 위해 개인 정보를 입력합니다.

1. **이메일 주소** 입력 또는 선택 (Microsoft 계정)
2. **국가/지역** 선택 (예: 대한민국)
3. **성명** 입력
4. **생년월일** 입력 (18세 이상이어야 함)
5. **약관 동의** 체크
6. **계속** 버튼 클릭

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/azure/azure-account-setup-step2-ko.png"
  alt="개인 정보 입력 화면"
  caption="정확한 개인 정보를 입력합니다"
/>

<DevTip>
국가/지역은 청구 주소와 일치해야 합니다. 나중에 변경할 수 있지만, 정확히 입력하는 것이 좋습니다.
</DevTip>

</Step>

---

<Step number={3} title="신원 인증 (카드 등록)">

유효한 신용카드 또는 체크카드로 신원을 확인합니다.

1. **카드 유형** 선택 (신용카드, 체크카드 등)
2. **카드 번호**, **만료 월/년**, **CVC** 입력
3. **카드 소유자 이름** 입력
4. **청구 주소** 입력 또는 확인
5. 약간의 금액(약 $1)이 임시로 승인되었다가 취소됨 (인증용)
6. **다음** 클릭

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/azure/azure-account-setup-step3-ko.png"
  alt="신용카드 등록 화면"
  caption="카드 정보가 안전하게 암호화되어 저장됩니다"
/>

<Callout type="info">
🔒 **보안 및 청구**

- Azure는 카드 정보를 **PCI DSS 표준**에 따라 암호화
- 자동 결제는 사용자가 명시적으로 활성화하지 않는 한 발생하지 않음
- $200 크레딧 또는 12개월이 지난 후 자동 결제 안 함
- 비용이 발생할 가능성이 있을 때 미리 알림
</Callout>

</Step>

---

<Step number={4} title="휴대폰 인증 (SMS)">

휴대폰 번호로 본인 확인을 완료합니다.

1. **국가 코드** 선택 (예: +82 대한민국)
2. **휴대폰 번호** 입력 (국번 제외)
3. **코드 전송 방식** 선택:
   - **SMS**: 문자 메시지로 받기
   - **전화**: 자동 음성 통화로 받기
4. **다음** 클릭
5. 수신한 **확인 코드** 입력
6. **코드 확인** 클릭

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/azure/azure-account-setup-step4-ko.png"
  alt="휴대폰 인증 화면"
  caption="SMS 또는 전화로 인증 코드를 받습니다"
/>

<Callout type="warning">
⚠️ **휴대폰 인증이 필요한 이유**

Azure는 실제 사용자임을 확인하기 위해 다중 인증을 사용합니다. 이는 계정 보안을 위한 필수 단계입니다.
</Callout>

</Step>

<Step number={5} title="Azure 계정 생성 완료">

축하합니다! 🎉 Azure 계정이 성공적으로 생성되었습니다.

1. Azure Portal 접속: [Azure Portal](https://portal.azure.com)
2. 좌측 "리소스 만들기" 메뉴로 서비스 생성 시작
3. **$200 크레딧** 확인:
   - 프로필 아이콘 > **비용 관리 + 청구**
   - **크레딧** 섹션에서 잔액 확인
4. **구독** 상세 정보 확인:
   - 구독 이름, ID, 상태 등
5. 알림 설정 (선택사항):
   - 지출 한도 설정
   - 비용 알림 설정

## 문제 해결

<DevTip>
**"신용카드 거절됨" 또는 "인증 실패"**

다음을 확인하세요:
1. 카드가 국제 거래를 지원하는지 확인 (은행에 문의)
2. 카드 정보가 정확하게 입력되었는지 재확인
3. 다른 카드로 시도
4. Azure 지원팀에 문의
</DevTip>

<Callout type="warning">
⚠️ **무료 크레딧 사용 팁**

1. **지출 한도 설정**: 실수로 인한 과다 청구 방지
2. **비용 모니터링**: 정기적으로 비용 확인
3. **리소스 정리**: 사용하지 않는 리소스 삭제
4. **가격 계산**: 시작 전 [가격 계산기](https://azure.microsoft.com/pricing/calculator/) 사용
5. **크레딧 종료**: 크레딧 종료 전에 결제 방법 등록 또는 리소스 삭제
</Callout>

## Azure 프리 티어 서비스

**12개월 무료 서비스:**
- Virtual Machines (1 vCPU, 1GB RAM)
- SQL Database (1GB 저장소)
- App Service (만 시간)
- Cosmos DB (25GB 저장소)
- Functions (100만 실행)
- 그 외 25개 이상의 서비스

**무기한 무료 서비스:**
- Static Web Apps
- Cognitive Search
- DevOps Projects
- 그 외 여러 서비스
