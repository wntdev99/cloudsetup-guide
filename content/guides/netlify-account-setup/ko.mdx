# Netlify 계정 만들기 - 웹사이트 배포 및 Git 통합 설정

Netlify는 **웹사이트를 배포하는 가장 쉬운 방법**입니다. GitHub 저장소를 연결하면 코드를 푸시할 때 Netlify가 자동으로 사이트를 배포합니다. 10분 안에 계정을 설정합니다.

## 시작하기 전에

<PrerequisiteCheck items={[
  { text: "이메일 주소", required: true },
  { text: "GitHub 계정 (선택, 권장)", required: false }
]} />

<Callout type="info">
🚀 **Netlify란?** Netlify는 웹사이트 호스팅 및 배포 플랫폼입니다. 정적 사이트 (HTML, CSS, JS), Next.js, React, Vue 등에 특히 좋습니다. 무료 요금제에는 월 100GB 대역폭이 포함됩니다.
</Callout>

---

<Step number={1} title="Netlify에 가입">

1. [Netlify.com](https://www.netlify.com)을 방문합니다.
2. **가입** 버튼을 클릭합니다 (우측 상단).
3. 가입 방법을 선택합니다:
   - 이메일과 비밀번호
   - GitHub 계정
   - GitLab 계정
   - Bitbucket 계정
4. 선호하는 가입 방법을 클릭합니다.
5. 이메일 사용 시: 이메일과 비밀번호를 입력한 후 **가입**을 클릭합니다.

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/netlify/step1-signup-ko.png"
  alt="Netlify 가입 페이지"
  caption="이메일 또는 Git 제공자로 Netlify 계정을 만드세요"
  priority={true}
/>

<Callout type="tip">
💡 **팁**: GitHub로 가입하면 원활한 Git 통합을 할 수 있습니다 - 나중에 GitHub을 별도로 인증할 필요가 없습니다.
</Callout>

</Step>

---

<Step number={2} title="이메일 확인 (이메일 가입을 사용한 경우)">

1. 이메일 가입을 사용한 경우 이메일 받은편지함에서 **Netlify 확인**을 확인합니다.
2. **확인 링크**를 클릭합니다.
3. Netlify로 리다이렉트됩니다 - 확인이 완료됩니다!

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/netlify/step2-verify-email-ko.png"
  alt="이메일 확인 화면"
  caption="확인 링크를 클릭하여 이메일을 확인합니다"
/>

<Callout type="info">
📧 **참고**: GitHub/GitLab/Bitbucket으로 가입한 경우 이메일 확인을 건너뛸 수 있습니다.
</Callout>

</Step>

---

<Step number={3} title="GitHub 인증 (아직 하지 않은 경우)">

1. Netlify 대시보드에서 **새 사이트 추가**를 클릭합니다.
2. **기존 프로젝트 가져오기**를 선택합니다.
3. Git 제공자로 **GitHub**을 선택합니다.
4. **Netlify 인증**을 클릭하여 GitHub 계정을 연결합니다.
5. GitHub 팝업이 나타납니다 - **인증**을 클릭하여 Netlify에 액세스 권한을 부여합니다.

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/netlify/step3-github-auth-ko.png"
  alt="GitHub 인증"
  caption="GitHub 저장소에 대한 Netlify 액세스 권한 부여"
/>

<Callout type="warning">
🔐 **권한**: Netlify가 저장소 액세스를 요청합니다. 이것은 정상입니다 - Netlify는 코드에 접근하여 배포해야 합니다.
</Callout>

</Step>

---

<Step number={4} title="Netlify 대시보드에 접근">

1. 인증 후 **Netlify 대시보드**에 있습니다.
2. 이제 다음을 할 수 있습니다:
   - 템플릿에서 새 사이트 생성
   - 기존 Git 저장소 배포
   - 파일을 수동으로 업로드
3. 고유한 **Netlify 하위 도메인**이 자동으로 할당됩니다.

## 자주 묻는 질문

<Callout type="info">
**Q. Netlify에 무료 요금제가 있나요?**

A. 네! 무료 요금제에는 월 100GB 대역폭, 무제한 사이트, 연속 배포가 포함됩니다. 신용카드가 필요 없습니다.
</Callout>

<Callout type="info">
**Q. 호스팅할 수 있는 사이트 유형은 무엇인가요?**

A. 정적 사이트 (HTML/CSS/JS), React, Vue, Next.js, Angular, Hugo, Jekyll 등입니다. 기본적으로 모든 프론트엔드 프레임워크 또는 정적 사이트입니다.
</Callout>

<Callout type="info">
**Q. 자신의 도메인을 사용할 수 있나요?**

A. 네! 모든 Netlify 사이트에 커스텀 도메인을 추가할 수 있습니다. Netlify는 무료 SSL 인증서를 제공합니다.
</Callout>

<Callout type="info">
**Q. Netlify는 어떤 Git 제공자를 지원하나요?**

A. GitHub, GitLab, Bitbucket, Azure DevOps입니다. 배포를 트리거하려면 이들 중 하나를 사용할 수 있습니다.
</Callout>

## 문제 해결

### 확인 이메일을 찾을 수 없습니다

- 스팸/정크 폴더를 확인합니다
- 이메일 배송을 위해 몇 분 기다립니다
- 가입 페이지에서 "확인 다시 보내기"를 클릭합니다
- 다른 이메일로 가입해봅시다

### GitHub 인증 실패

- 올바른 GitHub 계정을 사용하고 있는지 확인합니다
- GitHub 계정 이메일이 확인되었는지 확인합니다
- 계정 설정에서 연결을 끊고 다시 연결해봅시다
- 브라우저 캐시를 지우고 다시 시도합니다

### 가입 후 대시보드를 볼 수 없습니다

- 빈 페이지로 리다이렉트된 경우 새로고침을 시도합니다
- 로그인했는지 확인합니다 (프로필 아이콘 찾기)
- 쿠키를 지우고 다시 로그인해봅시다
- 테스트를 위해 시크릿/비공개 브라우징을 사용합니다
