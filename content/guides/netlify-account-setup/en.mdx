<Step number={1} title="Sign Up for Netlify">

1. Visit [Netlify.com](https://www.netlify.com).
2. Click **Sign Up** button (top right).
3. Choose how to sign up:
   - Email and password
   - GitHub account
   - GitLab account
   - Bitbucket account
4. Click your preferred sign-up method.
5. If using email: enter email and password, then click **Sign Up**.

<Step number={3} title="Authorize GitHub (if not already done)">

1. In your Netlify dashboard, click **Add new site**.
2. Select **Import an existing project**.
3. Choose **GitHub** as your Git provider.
4. Click **Authorize Netlify** to connect your GitHub account.
5. A GitHub popup appears - click **Authorize** to grant Netlify access.

## Troubleshooting

### Can't find verification email

- Check spam/junk folder
- Wait a few minutes for email delivery
- Click "Resend verification" on the signup page
- Try signing up with a different email

### GitHub authorization failed

- Make sure you're using the correct GitHub account
- Check that your GitHub account email is verified
- Try disconnecting and reconnecting in account settings
- Clear browser cache and try again

### Can't see dashboard after signup

- If redirected to a blank page, try refreshing
- Check that you're logged in (look for your profile icon)
- Try clearing cookies and logging in again
- Use incognito/private browsing to test
