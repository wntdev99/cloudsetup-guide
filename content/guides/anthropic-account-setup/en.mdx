<Step number={4} title="Set API Key as Environment Variable">

### Set Environment Variable

Store API key as environment variable.

#### macOS/Linux

<CopyBlock language="bash" title="Set environment variable">
export ANTHROPIC_API_KEY="sk-ant-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
</CopyBlock>

Or permanently in `~/.bashrc` or `~/.zshrc`:

<CopyBlock language="bash" title="Add to .bashrc or .zshrc">
echo 'export ANTHROPIC_API_KEY="sk-ant-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"' >> ~/.bashrc
source ~/.bashrc
</CopyBlock>

#### Windows (PowerShell)

<CopyBlock language="powershell" title="Set in PowerShell">
$env:ANTHROPIC_API_KEY="sk-ant-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
</CopyBlock>

Or permanently:

<CopyBlock language="powershell" title="Set permanently">
[System.Environment]::SetEnvironmentVariable(
  'ANTHROPIC_API_KEY',
  'sk-ant-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx',
  'User'
)
</CopyBlock>

### Use .env File (Recommended)

Use `.env` file in your project:

<CopyBlock language="bash" title=".env file">
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
</CopyBlock>

Load with python-dotenv or similar:

<CopyBlock language="python" title="Load .env in Python">
from dotenv import load_dotenv
import os

load_dotenv()
api_key = os.getenv('ANTHROPIC_API_KEY')
</CopyBlock>

## Claude Model Information

Claude comes in several versions:

### Claude 3 Opus (Most Powerful)
- Best for complex tasks
- Highest latency
- $15 per 1M input tokens

### Claude 3 Sonnet (Balanced)
- Recommended for most tasks
- Mid-range pricing
- $3 per 1M input tokens

### Claude 3 Haiku (Fastest)
- Fast response needed
- Most affordable
- $0.80 per 1M input tokens

## Billing Information

### Pricing Structure

Claude API charges based on tokens:

<CopyBlock language="text" title="Pricing example">
Process 1,000 input tokens + generate 200 output tokens

Claude 3 Haiku pricing:
Input: 1,000 √ó ($0.80 / 1,000,000) = $0.0008
Output: 200 √ó ($2.40 / 1,000,000) = $0.00048
Total: approximately $0.0013
</CopyBlock>

### Free Trial Policy

- Anthropic currently offers no free trial
- Payment required to start immediately
- Creating API key without usage incurs no charges

1. Go to **Billing** tab in Anthropic console
2. Check **Usage**
3. Track monthly costs

## Security Tips

### Protect API Keys

1. Never commit to public repositories
2. Add `.env` file to `.gitignore`
3. Delete unused keys regularly
4. Use separate keys for team members

### Rate Limits

Each API key has rate limits:
- Default: hundreds of requests per second
- High usage: request increase through priority support

## Troubleshooting

### API Key Not Working

1. Verify API key copied correctly
2. Check environment variable is set properly
3. Verify API key isn't disabled

### Payment Error

1. Verify card info is accurate
2. Check card expiration
3. Confirm auto-pay is enabled

### Usage Higher Than Expected

1. Check logs for unnecessary API calls
2. Improve token efficiency with batching
3. Use caching to prevent duplicate requests
