## What is Docker?

<Callout type="info">
üê≥ **Docker is a containerization platform**

Key features of Docker:
- **Environment consistency**: Run the same environment as development
- **Fast deployment**: Much faster startup than virtual machines
- **Resource efficiency**: Each container is lightweight and independent
- **Version control**: Precise version management through images
- **Scalability**: Easily manage multiple containers

Examples:
- `nodejs-app`: For Node.js applications
- `postgres-db`: For PostgreSQL database
- `nginx-server`: For Nginx web server
</Callout>

<Step number={1} title="Download Docker Desktop">

### macOS Installation

1. Visit [Docker official website](https://www.docker.com/products/docker-desktop).
2. Click the **Download for Mac** button at the top.
3. Select based on your Mac's CPU:

## Troubleshooting

### "Cannot connect to Docker daemon"

- Verify Docker Desktop is running
- Check the menu bar for the Docker icon on Mac/Windows
- Try restarting Docker

### "Permission denied while trying to connect to Docker daemon" [Linux]

- Add user to docker group: `sudo usermod -aG docker $USER`
- Restart your terminal or run `newgrp docker`

### Windows: "WSL 2 installation is incomplete"

- Run Windows PowerShell as administrator
- Run `wsl --install`
- Restart your system
