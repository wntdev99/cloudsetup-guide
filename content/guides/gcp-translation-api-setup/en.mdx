<Step number={2} title="Prepare API Key">

To use Translation API, you need an **API key** or **service account**.

### 2-1. Reuse Existing API Key (if you have a Vision API key)

If you've already created an API key for Vision API or other GCP APIs:

1. Click **Credentials** in the left menu
2. Check existing API keys
3. Verify **Cloud Translation API** is included in the key's **API restrictions** section
4. If not included:
   - Click the API key name
   - Select **Restrict key** in the **API restrictions** section
   - Check the **Cloud Translation API** checkbox
   - Click **Save**

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/gcp-translation-api-step2a-en.png"
  alt="Add Translation API to existing key"
  caption="Add Translation API to API restrictions"
/>

### 2-2. Create New API Key (if you don't have one)

1. Click **Credentials** in the left menu
2. Click **+ CREATE CREDENTIALS** button at the top
3. Select **API key** from the dropdown

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/gcp-translation-api-step2b-en.png"
  alt="Create API key"
  caption="Create a new API key"
/>

4. **Copy** the generated API key and save it somewhere safe.
5. Click **Restrict Key** button to configure security:
   - **API restrictions**: Select "Restrict key"
   - Check only **Cloud Translation API**
   - Click **Save**

## Frequently Asked Questions

<Callout type="info">
**Q. Which languages are supported?**

A. 100+ languages are supported. Major languages:
- Korean (ko), English (en), Japanese (ja), Chinese Simplified (zh-CN), Chinese Traditional (zh-TW)
- Spanish (es), French (fr), German (de), Russian (ru), Arabic (ar)
- Hindi (hi), Vietnamese (vi), Thai (th), Indonesian (id), etc.

Full list: Available via `languages` API endpoint
</Callout>

<Callout type="info">
**Q. How is the translation quality?**

A. Uses the same engine as Google Translate:
- General text: Very high quality
- Technical terms: Can be improved with glossary feature (advanced)
- Context understanding: More accurate with longer sentences
- Real-time conversation: Sufficient for practical use
</Callout>

<Callout type="info">
**Q. How are characters counted?**

A. Counts all characters included in the request:
- Spaces, special characters, and line breaks included
- HTML tags excluded (when format: "html")
- Multiple text array requests count all texts combined

Examples:
- "Hello World" = 11 characters (space included)
- ["Hi", "Bye"] = 5 characters
</Callout>

<Callout type="info">
**Q. How do I know the language codes?**

A. Uses ISO 639-1 standard:
- `ko`: Korean
- `en`: English
- `ja`: Japanese
- `zh-CN`: Chinese Simplified
- `zh-TW`: Chinese Traditional
- `es`: Spanish
- `fr`: French
- `de`: German

Full list available via `/languages` API or [official documentation](https://cloud.google.com/translate/docs/languages).
</Callout>

## Troubleshooting

### "API key not valid" error

- Verify API key was copied correctly
- Check Translation API is allowed in key restrictions
- Verify key is not disabled

### "Permission denied" error

- Confirm Translation API is enabled
- Verify working in correct project
- Check billing account is linked

### Translation results are strange

- Verify source text is correct
- Try explicitly specifying `source` language:
  ```json
  {
    "q": "text",
    "source": "en",
    "target": "ko"
  }
  ```
- Longer texts reflect context better
- Use glossary feature for technical terms (advanced)

### No response or slow

- Check network connection
- Verify API key quota not exceeded
- Check request size (max 102,400 characters/request)
