# DigitalOcean 계정 만들기 - 클라우드 서버 호스팅

DigitalOcean은 **개발자를 위한 클라우드 플랫폼**으로서 호스팅을 간단하고 저렴하게 만듭니다. 새 사용자는 **60일 동안 유효한 $200 크레딧**을 받습니다. 15분 안에 계정을 설정합니다.

## 시작하기 전에

<PrerequisiteCheck items={[
  { text: "이메일 주소", required: true },
  { text: "이메일로 검증", required: true }
]} />

<Callout type="info">
☁️ **DigitalOcean이란?** DigitalOcean은 가상 서버 (Droplet), 데이터베이스, 앱 호스팅 및 기타 클라우드 서비스를 제공합니다. 단순함과 훌륭한 문서로 알려져 있습니다 - 개발자와 스타트업에 완벽합니다.
</Callout>

<Step number={1} title="DigitalOcean에 가입">

1. [DigitalOcean.com](https://www.digitalocean.com)을 방문합니다.
2. **가입** 버튼을 클릭합니다 (우측 상단).
3. 가입 방법을 선택합니다:
   - 이메일과 비밀번호
   - Google 계정
   - GitHub 계정
4. 이메일을 입력하거나 소셜 로그인을 선택합니다.
5. **가입** 또는 **Google/GitHub와 계속하기**를 클릭합니다.

/>

<Callout type="tip">
💡 **팁**: GitHub 또는 Google로 가입하면 더 빠릅니다.
</Callout>

</Step>

---

<Step number={2} title="이메일 확인">

1. **DigitalOcean 확인 메시지**에 대한 이메일을 확인합니다.
2. 이메일의 **확인 링크**를 클릭합니다.
3. DigitalOcean으로 리다이렉트됩니다.
4. 확인이 완료됩니다!

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/digitalocean/step2-verify-email-ko.png"
  alt="이메일 확인 화면"
  caption="확인 링크를 클릭하여 이메일을 확인합니다"
/>

<Callout type="warning">
⚠️ **참고**: 확인 이메일이 도착하지 않으면 스팸 폴더를 확인합니다. 필요하면 새로운 확인 이메일을 요청합니다.
</Callout>

</Step>

---

<Step number={3} title="결제 정보 추가 및 $200 크레딧 청구">

1. **프로필 아이콘** (우측 상단)을 클릭합니다.
2. **결제** 섹션으로 이동합니다.
3. **결제 정보**를 추가합니다 (무료 크레딧은 선택사항).
4. 프로모 코드가 있으면 **프로모 코드**를 입력합니다.
5. 새 계정의 경우 **$200 무료 크레딧**이 자동으로 나타나야 합니다.
6. 크레딧은 **60일** 동안 유효합니다.

  src="https://res.cloudinary.com/demo/image/upload/v1/digitalocean/step3-billing-credits-ko.png"

/>

<Callout type="info">
💰 **무료 크레딧**: 새 사용자는 자동으로 60일 동안 유효한 $200 크레딧을 받습니다. 그 후에는 삭제하지 않으면 사용하는 리소스에 대해 비용이 청구됩니다.
</Callout>

</Step>

---

<Step number={4} title="DigitalOcean 대시보드에 접근">

1. **프로필 아이콘** (우측 상단)을 클릭합니다.
2. **제어판** (또는 **대시보드**)으로 이동합니다.
3. 이제 Droplet, 데이터베이스 및 기타 리소스를 생성할 수 있습니다.
4. 대시보드는 크레딧 잔액을 표시합니다.

## 자주 묻는 질문

<Callout type="info">
**Q. $200 크레딧은 얼마나 유효한가요?**

A. $200 크레딧은 계정 생성 후 60일 동안 유효합니다. 그 후에는 만료되지만 만든 모든 리소스는 계속 실행되고 비용이 청구됩니다.
</Callout>

<Callout type="info">
**Q. 60일 후에는 어떻게 되나요?**

A. 60일 후에는 크레딧이 만료됩니다. 실행 중인 리소스가 있으면 DigitalOcean 가격 책정에 따라 비용이 청구됩니다. 기본 $5/월 Droplet의 비용은 월 $5입니다.
</Callout>

<Callout type="info">
**Q. 신용카드가 필요한가요?**

A. $200 크레딧을 통한 무료 체험의 경우 일반적으로 신용카드가 필요하지 않습니다. 그러나 DigitalOcean은 검증을 위해 결제 정보를 요청할 수 있습니다. 무료 크레딧이 만료된 후에는 유효한 결제 수단이 필요합니다.
</Callout>

<Callout type="info">
**Q. 프로덕션에 DigitalOcean을 사용할 수 있나요?**

A. 예! 많은 프로덕션 애플리케이션이 DigitalOcean에서 실행됩니다. 저렴하고 신뢰할 수 있으며 좋은 문서가 있습니다. 프로덕션의 경우 무료 크레딧이 만료된 후 유료 요금제가 필요합니다.
</Callout>

## 문제 해결

### 확인 이메일을 받지 못했습니다

- 스팸/정크 폴더를 확인합니다
- 이메일 배송을 위해 잠시 기다립니다
- 가입 양식에서 "확인 이메일 다시 보내기"를 클릭합니다
- 다른 이메일 주소로 가입해봅시다

### 크레딧이 표시되지 않습니다

- 결제 섹션으로 이동하여 크레딧을 확인합니다
- 새 사용자 크레딧은 자동으로 나타납니다 - 몇 분이 걸릴 수 있습니다
- 10분 후에도 보이지 않으면 DigitalOcean 지원팀에 문의합니다

### 제어판에 접근할 수 없습니다

- 먼저 이메일이 확인되었는지 확인합니다
- 로그아웃 후 다시 로그인해봅시다
- 브라우저 캐시와 쿠키를 지웁니다
- 다른 브라우저를 시도해봅시다
