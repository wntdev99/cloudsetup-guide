# Pinecone 계정 만들기 - AI 검색을 위한 벡터 데이터베이스

Pinecone은 AI 애플리케이션을 위해 설계된 클라우드 네이티브 벡터 데이터베이스입니다. 빠른 의미 검색, 추천 시스템, 임베딩 기반 기능을 가능하게 합니다. 15분 안에 계정을 설정해봅시다!

## Pinecone이란?

<Callout type="info">
🔍 **Pinecone은 의미 검색을 가능하게 합니다**

- **벡터 저장소**: 고차원 벡터 저장 및 검색
- **빠른 검색**: 밀리초 단위로 유사 항목 찾기
- **확장 가능**: 수천 개부터 수십억 개 벡터까지 성장
- **프로덕션 준비**: Fortune 500 회사에서 사용
- **AI 통합**: OpenAI, Hugging Face의 임베딩과 완벽하게 작동
- **서버리스**: 인프라 관리 불필요

이렇게 사용됩니다:
- 의미 검색
- 추천 엔진
- 유사성 매칭
- 문서 검색
- 이미지 검색
</Callout>

## 문제 해결

### 이메일을 확인할 수 없음

- 스팸/정크 폴더 확인
- 확인 재전송 시도
- 다른 이메일 사용
- support@pinecone.io로 문의

### 차원 불일치 오류

```
ValueError: expected dimension 1536 but got 384
```

해결책:
- 인덱스 차원과 일치하는 임베딩 모델 사용
- 올바른 차원으로 새 인덱스 만들기
- 차원과 일치하도록 임베딩 모델 변경

### API 키가 작동하지 않음

- 키가 완전히 복사되었는지 확인
- .env 파일에 올바른 변수명이 있는지 확인
- 키가 올바른 환경인지 확인
- 확실하지 않으면 키 재생성
