## Understanding Vector Embeddings

<CopyBlock
  code="What are Vector Embeddings?

Numbers Representation:
  Text/Image -> AI Model -> Numbers (Vector)
  Example: 'dog' -> [0.5, -0.3, 0.8, ...]

Dimension:
  Each embedding has fixed size
  OpenAI: 1536 dimensions
  Hugging Face: 384, 768, or 1536 dimensions

Similarity:
  Vectors close in space = similar meaning
  dog [0.5, -0.3, 0.8]
  puppy [0.52, -0.31, 0.79]  -> Very similar!
  cat [0.3, 0.5, -0.2]       -> Different"

## Troubleshooting

### Can't verify email

- Check spam/junk folder
- Try resending verification
- Use alternative email
- Contact support@pinecone.io

### Dimension mismatch error

```
ValueError: expected dimension 1536 but got 384
```

Solutions:
- Use embedding model matching your index dimension
- Create new index with correct dimension
- Change embedding model to match dimension

### API key not working

- Verify key is copied completely
- Check .env file has correct variable name
- Ensure key is for correct environment
- Regenerate key if unsure
