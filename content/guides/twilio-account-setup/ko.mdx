# Twilio 계정 만들기 및 전화번호 설정

Twilio는 **SMS, 음성 통화, 비디오 등의 통신 기능을 애플리케이션에 추가하는 강력한 플랫폼**입니다. 무료로 시작하여 전 세계 사용자와 통신할 수 있습니다. 20분 안에 계정을 설정하고 첫 SMS를 보낼 준비를 할 수 있습니다.

</Callout>

<Step number={1} title="Twilio 계정 생성">

1. [Twilio.com](https://www.twilio.com)을 방문합니다.
2. **가입하기** 또는 **Sign Up** 버튼을 클릭합니다.
3. **이메일 주소**, **비밀번호**, **이름**을 입력합니다.
4. **약관에 동의**하고 **계정 생성**을 클릭합니다.

/>

<Callout type="tip">
💡 **팁**: GitHub, Google, Microsoft 계정으로도 가입할 수 있습니다.
</Callout>

</Step>

---

<Step number={2} title="계정 정보 입력 및 전화번호 확인">

1. **사용 사례** 선택 (예: SMS 전송, 음성 통화 등).
2. **이름**, **회사**, **국가** 입력.
3. **전화번호** 입력 및 확인:
   - SMS로 인증 코드를 받거나
   - 전화 통화로 확인

4. 인증 코드를 입력합니다.

  src="https://res.cloudinary.com/demo/image/upload/v1/twilio/step2-phone-verification-ko.png"

<Callout type="warning">
⚠️ **중요**: Twilio는 계정 보안을 위해 전화번호 확인을 필수로 요구합니다.
</Callout>

</Step>

<Step number={3} title="무료 평가판 크레딧 확인 및 API 키 생성">

1. 대시보드로 이동합니다.
2. **Account Settings**에서 무료 평가판 $15 크레딧을 확인합니다.
3. **API Keys & Tokens** 섹션으로 이동합니다:
   - **Account SID** 복사
   - **Auth Token** 복사 (절대 공개하지 마세요!)

4. 또는 새 API 키를 생성합니다:
   - **Create API Key** 버튼 클릭
   - **프렌들리 이름** 입력 (예: My App API Key)
   - **SID 타입** 선택 (Standard)
   - **Create** 클릭

<Callout type="warning">
⚠️ **보안**: Auth Token을 절대 공개하지 마세요. 코드에 하드코딩하지 말고 환경 변수에 저장하세요.

</Step>

<Step number={4} title="Twilio 전화번호 획득">

SMS 또는 음성 통화를 보내려면 Twilio 전화번호가 필요합니다.

**평가판 전화번호 获得:**

1. 대시보드에서 **Phone Numbers**를 클릭합니다.
2. **Get Started**를 클릭합니다.
3. **Buy a Number**를 선택합니다.
4. **국가** 선택 (예: 미국 +1).
5. **Area Code** 입력 (선택사항).
6. 원하는 **전화번호** 선택.
7. **Buy** 또는 **구매**를 클릭합니다.

**환경 변수 설정:**

.env 파일에 저장:

<CopyBlock
  code="TWILIO_ACCOUNT_SID=AC...
TWILIO_AUTH_TOKEN=your_auth_token_here
TWILIO_PHONE_NUMBER=+1234567890"
  language="bash"
/>

**검증된 발신자 추가 (평가판용):**

평가판 모드에서는 최대 20개 검증된 전화번호에만 SMS를 보낼 수 있습니다.

1. **Verified Caller IDs**로 이동합니다.
2. **Add a Verified Caller ID**를 클릭합니다.
3. 수신자 전화번호를 입력합니다.
4. SMS로 받은 코드를 입력하여 확인합니다.

<DevTip>
**평가판 모드 제한사항:**

- 최대 $15 크레딧 사용 가능
- SMS는 검증된 수신자에게만 발송 (최대 20개)
- 제한 없음 (크레딧 범위 내)

프로덕션을 위해서는 유료 계정으로 업그레이드해야 합니다.

## 자주 묻는 질문

<Callout type="info">
**Q. Twilio 전화번호의 월 비용은 얼마인가요?**

A. 국가에 따라 다르지만, 보통 $1~$2/월입니다. 평가판 $15 크레딧으로 여러 개월 사용할 수 있습니다.
</Callout>

<Callout type="info">
**Q. 평가판 크레딧이 다 떨어지면?**

A. 추가 서비스를 사용하려면 신용카드를 등록하고 유료로 전환해야 합니다. 크레딧 한도 설정으로 비용을 제어할 수 있습니다.
</Callout>

<Callout type="info">
**Q. Auth Token을 잃어버렸습니다**

A. 대시보직에서 새 Auth Token을 생성하면 됩니다. 이전 토큰은 자동으로 무효화됩니다.
</Callout>

<Callout type="info">
**Q. 어떤 국가에서 Twilio를 사용할 수 있나요?**

A. 거의 모든 국가에서 사용 가능합니다. [Twilio 지원 국가 목록](https://www.twilio.com/docs/phone-numbers/phone-number-capabilities)을 확인하세요.
</Callout>

## 문제 해결

### 평가판 계정으로 제한됨

- 상태를 확인합니다: `Account > General Settings > Account Type`
- 신용카드를 등록합니다
- Twilio에 업그레이드를 요청합니다

### SMS 전송이 실패합니다

- 수신자 번호가 검증되었는지 확인합니다 (평가판)
- Account SID와 Auth Token이 올바른지 확인합니다
- 크레딧이 남아있는지 확인합니다

### 전화번호를 구매할 수 없습니다

- 결제 정보를 입력했는지 확인합니다
- 해당 국가에서 Twilio 서비스를 지원하는지 확인합니다
