<Step number={1} title="Connect Your GitHub Repository to Netlify">

1. Log in to **Netlify Dashboard**.
2. Click **Add new site** button.
3. Select **Import an existing project**.
4. Choose **GitHub** as your Git provider.
5. Select your **repository** from the list.
6. Click **Deploy site**.

<Step number={2} title="Configure Build Settings">

Netlify will auto-detect your build settings based on your project. Review them:

1. **Base directory**: Usually blank (root of repo).
2. **Build command**:
   - React/Next.js: `npm run build`
   - Static HTML: Leave blank
   - Hugo: `hugo`
3. **Publish directory**:
   - React/Next.js: `out` or `build`
   - Static HTML: `public` or `.`
4. Click **Deploy site**.

## Frequently Asked Questions

<Callout type="info">
**Q. How often are deployments triggered?**

A. Every time you push to the main branch. Each push triggers a new build and deployment within seconds.
</Callout>

<Callout type="info">
**Q. Can I test changes before deploying?**

A. Yes! Netlify creates **preview deployments** for pull requests. You can test your changes before merging to main.
</Callout>

<Callout type="info">
**Q. What if a deployment fails?**

A. You can **instantly rollback** to a previous deployment. Just click "Trigger deploy" on any previous build in your deployment history.
</Callout>

<Callout type="info">
**Q. Can I deploy from other branches?**

A. Yes! Configure different branches to deploy to different URLs. Great for staging environments.
</Callout>

## Troubleshooting

### Build failed - needs npm dependencies

- Make sure `package.json` is in your repository root
- Ensure all dependencies are listed (npm install works locally)
- Check build logs for specific error messages

### Site shows 404 error after deploy

- Verify your publish directory is correct
- Check that the build output directory exists
- For React/Next.js, confirm build command ran successfully

### Automatic deployment didn't trigger

- Verify GitHub integration is still authorized
- Check that you pushed to the correct branch (main/master)
- Look at "Deploys" tab for push event history
- Try manually triggering deployment: click "Trigger deploy"

### HTTPS not working

- Wait a few minutes - SSL certificate generation takes time
- Go to Domain settings and check HTTPS status
- If manual domain, ensure DNS is pointing to Netlify
