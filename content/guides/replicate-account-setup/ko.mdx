# Replicate 계정 설정하기

**Replicate**는 수백 개의 오픈소스 AI 모델에 쉽게 접근할 수 있는 API입니다. 이미지 생성, 텍스트 처리, 음성 인식 등 다양한 AI 기능을 사용할 수 있습니다. 이 가이드를 따라 10분 안에 계정을 설정하고 첫 번째 API 키를 발급받으세요.

<Step number={1} title="Replicate 웹사이트 방문">

### 콘솔 접근

1. [replicate.com](https://replicate.com)을 방문합니다
2. 우측 상단의 **Sign up** 버튼을 클릭합니다
3. 회원가입 옵션을 확인합니다

### 회원가입 방법 선택

Replicate은 다음 방법으로 가입할 수 있습니다:

1. **GitHub로 가입** - GitHub 계정이 있으면 가장 빠릅니다
2. **Google로 가입** - Google 계정 사용
3. **이메일로 가입** - 이메일 주소와 비밀번호 사용

<Step number={2} title="계정 정보 입력">

### GitHub으로 가입하기 (추천)

1. **Sign up with GitHub** 버튼 클릭
2. GitHub 로그인 페이지로 이동
3. GitHub 계정으로 로그인
4. Replicate이 GitHub 프로필에 접근하는 것을 승인 클릭

### 이메일로 가입하기

이메일 옵션을 선택한 경우:

1. 이메일 주소 입력
2. 강력한 비밀번호 설정
3. 이용약관 동의
4. **Create account** 클릭

<Step number={3} title="이메일 확인">

### 이메일 인증

1. 가입에 사용한 이메일 받은편지함 확인
2. Replicate 확인 이메일 찾기
3. 이메일의 **확인[Verify]** 버튼 또는 링크 클릭
4. 자동으로 Replicate 대시보드로 이동

### 확인 완료

성공적으로 인증되면:

1. Replicate 대시보드가 표시됩니다
2. 무료 크레딧 정보가 표시됩니다
3. 프로필 설정이 완료됩니다

<Step number={4} title="API 키 생성">

### API 키 페이지 접근

1. Replicate 대시보드에서 우측 상단의 **계정[Account]** 클릭
2. **설정[Settings]** 또는 **API tokens** 선택
3. **API tokens** 또는 **API keys** 섹션으로 이동

### API 키 생성

1. **Create API token** 또는 **New API token** 버튼 클릭
2. 토큰 이름 입력 (선택사항, 예: "My Application")
3. **Generate** 또는 **Create** 클릭
4. 생성된 API 키가 표시됩니다

## 문제 해결

### API 키가 작동하지 않습니다

1. API 키가 올바르게 복사되었는지 확인
2. 환경 변수가 올바르게 설정되었는지 확인
3. API 키 복사 시 공백이 포함되지 않았는지 확인
4. 대시보드에서 토큰이 활성화되어 있는지 확인

### 크레딧이 모두 소진되었습니다

1. 결제 정보 추가하기
2. 대시보드의 **Billing** 섹션에서 결제 방법 등록
3. 자동 충전 설정 (선택사항)

### 모델에 접근할 수 없습니다

1. API 키가 올바른지 다시 확인
2. 모델이 공개 모델인지 확인
3. 네트워크 연결 확인
4. Replicate 상태 페이지 확인
