<Step number={4} title="Generate API Key">

3. Navigate to **API tokens** or **API keys** section

1. Click **Create API token** or **New API token** button
2. Enter token name (optional, e.g., "My Application")
3. Click **Generate** or **Create**
4. Your new API key displays

## Set Up API Key as Environment Variable

### Local Development

#### Create .env File

<CopyBlock language="bash" title=".env file">
REPLICATE_API_TOKEN=r8_Abc1XyZqWvT2FgHiJkL3MnOpQrStU4VwX5YzA6BcDeF7
</CopyBlock>

#### Use in Python

<CopyBlock language="python" title="Python Example">
import os
from dotenv import load_dotenv

load_dotenv()
api_token = os.getenv('REPLICATE_API_TOKEN')
</CopyBlock>

#### Use in Node.js

<CopyBlock language="javascript" title="Node.js Example">
const token = process.env.REPLICATE_API_TOKEN;
const replicate = new Replicate({
  auth: token,
});
</CopyBlock>

### Set Environment Variable (macOS/Linux)

<CopyBlock language="bash" title="Set Permanent Environment Variable">

source ~/.bashrc
</CopyBlock>

### Set Environment Variable (Windows PowerShell)

<CopyBlock language="powershell" title="PowerShell Environment Variable">
[System.Environment]::SetEnvironmentVariable(
  'REPLICATE_API_TOKEN',
  'r8_Abc1XyZqWvT2FgHiJkL3MnOpQrStU4VwX5YzA6BcDeF7',

## Troubleshooting

### API Key is not working

1. Verify API key was copied correctly
2. Check environment variable is set properly
3. Ensure no whitespace was included when copying
4. Confirm token is enabled in dashboard

### Credits exhausted

1. Add payment information
2. Go to **Billing** section in dashboard
3. Register a payment method
4. Set up auto-recharge (optional)

### Cannot access model

1. Double-check API key is correct
2. Confirm model is public
3. Check network connection
4. Check Replicate status page
