<Step number={1} title="Create SendGrid Account">

1. Visit [SendGrid.com](https://sendgrid.com).
2. Click the **Sign up** button.
3. Enter your **email address**.
4. Set a **password** (strong password recommended).
5. Enter your **first name** and **last name**.
6. Optionally enter **company name**.
7. Agree to **terms** and click **Create Account**.

<Step number={3} title="Generate API Key">

You need an API key to send emails from SendGrid.

1. In the dashboard, go to the **Settings** section in the left menu.
2. Click **API Keys** or **API Keys**.
3. Click the **Create API Key** button.
4. Enter an **API Key name** (e.g., "My App API Key").
5. Select the **permission level**:
   - **Restricted**: Grant specific permissions (recommended)
   - **Full Access**: Grant all permissions

6. **Required permissions**:
   - Mail Send
   - Contacts Import
   - Bounce (optional)

7. Click **Create & View** button.

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/sendgrid/step3-api-key-en.png"

**Copy and save the API key in a safe place:**

Your API key is displayed **only once**. If you lose it, you'll need to create a new one.

<Callout type="warning">
‚ö†Ô∏è **Security**: Never share your API key or commit it to version control. Store it in environment variables.

<Step number={4} title="Set Up Sender Verification">

You must verify the sender to send emails from SendGrid.

**Simple Verification (Recommended - Fast):**

1. In the dashboard, go to **Settings** menu.
2. Click **Sender Authentication** or **Sender Authentication**.
3. Choose **Single Sender Verification**.
4. Enter your **sender email address** (e.g., noreply@example.com).
5. Enter your **sender name** (e.g., My App).
6. Optionally enter a **reply-to email**.
7. Click **Create**.
8. SendGrid sends a **verification email**.
9. Click the **verification link** in the email.

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/sendgrid/step4-sender-verification-en.png"
  alt="SendGrid Sender Verification"
  caption="Verify your sender email address"
/>

**Domain Authentication (For Production - Recommended):**

Authenticating your domain provides higher deliverability.

1. On the **Sender Authentication** page, choose **Domain Authentication**.
2. Enter your **domain** (e.g., example.com).
3. Select **Default to Domain** (recommended).
4. **DNS records** will be displayed:
   - **CNAME** records (3)
   - Or **MX** records

5. Log in to your **DNS hosting provider**:
   - GoDaddy, Namecheap, Route 53, etc.
6. Add the records to your **DNS settings**.
7. Click the **Verify** button in SendGrid.
8. Once DNS records are verified, you're done.

## Developer Tips

<DevTip>
**Set API Key Environment Variables:**

Save in your .env file:

```
SENDGRID_API_KEY=SG.xxxxxxxxxxxxxxxxxxxx
SENDGRID_FROM_EMAIL=noreply@example.com
SENDGRID_FROM_NAME=My App
```

Access in Node.js:

```javascript
const apiKey = process.env.SENDGRID_API_KEY;
const fromEmail = process.env.SENDGRID_FROM_EMAIL;
```

## Frequently Asked Questions

<Callout type="info">
**Q. What happens if I exceed 100 emails per month?**

A. Your emails won't be sent. To send more, upgrade to a paid SendGrid plan.
</Callout>

<Callout type="info">
**Q. I lost my API key. What should I do?**

A. Just create a new API key. Click the "Create API Key" button on the API Keys page in your dashboard.
</Callout>

<Callout type="info">
**Q. What's the difference between sender verification and domain authentication?**

A. Sender verification confirms a specific email address and is quick. Domain authentication confirms your entire domain and provides higher deliverability.
</Callout>

<Callout type="info">
**Q. How can I prevent my emails from being marked as spam?**

A. Add SPF, DKIM, and DMARC records to your DNS. SendGrid automatically provides these records when you authenticate your domain.
</Callout>

## Troubleshooting

### Verification email doesn't arrive

- Check your spam folder
- Verify you used the correct email address
- Request verification email again
- Contact SendGrid support

### API key doesn't work

- Verify the API key was copied correctly
- Check that the API key has "Mail Send" permission
- Generate a new API key

### Sender verification failed

- Verify you used a valid email address
- Try domain authentication
- Contact SendGrid support
