<Step number={3} title="Setup Billing Information [Optional]">

### Add Payment Method

1. Go to account settings
2. Click **Billing** or **Payment** tab

Control costs by setting limits:

1. Go to **Billing** tab
2. **Usage limits** section
3. **Soft limit**: Alert amount [e.g., $10]
4. **Hard limit**: Stop API amount [e.g., $20]

<Step number={4} title="Generate and Secure API Keys">

### Generate API Key

1. Login to [OpenAI API Dashboard](https://platform.openai.com)
2. Click **API keys** or **Billing and usage** in left menu
3. Click **Create new secret key** or **+Create new secret key** button

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/openai/openai-account-setup-step4-en.png"
  alt="Generate API key"
  caption="Create API key in OpenAI dashboard"
/>

### Copy and Save API Key

1. Generated API key displays
2. Click **Copy** button to copy
3. Save in secure place [e.g., password manager]

<Callout type="warning">
‚ö†Ô∏è **API Key Security Precautions**

- Never share API key publicly
- Don't hardcode in application
- Save in .env file
- Never commit to GitHub
- Regenerate immediately if exposed

Wrong example:
```
const apiKey = 'sk-xxxxx'; // NEVER!
```

Right example:
```
const apiKey = process.env.OPENAI_API_KEY;
```
</Callout>

## Troubleshooting

### "Authentication failed"

- Verify API key is correct
- Verify API key was copied
- Check for "sk-" prefix

### "Rate limit exceeded"

- Limit API call frequency
- Check usage limits
- Wait before retrying

### Key not working

- Regenerate API key
- Create new .env file
- Reload environment variables
