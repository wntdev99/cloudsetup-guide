## OpenAI API란?

<Callout type="info">
🤖 **OpenAI API는 생성형 AI 모델에 접근하는 인터페이스입니다**

주요 기능:
- **Chat Completion**: ChatGPT 같은 대화형 AI
- **Text Generation**: 텍스트 생성 및 완성
- **Image Generation**: DALL-E 이미지 생성
- **Embedding**: 텍스트 벡터화
- **Speech**: 음성 합성 및 인식

인기 모델:
- `gpt-4`: 최신 고성능 모델
- `gpt-3.5-turbo`: 가성비 최고 모델
- `dall-e-3`: 이미지 생성
- `whisper`: 음성 인식
</Callout>

다음 중 하나를 선택합니다:

<Step number={2} title="이메일 인증 및 정보 입력">

### 이메일 인증

1. OpenAI에서 보낸 인증 이메일을 확인합니다.
2. **Verify email address** 링크를 클릭합니다.
3. 인증 완료 후 프로필 정보를 입력합니다.

### 프로필 정보

다음 정보를 입력합니다:

1. **First Name**: 이름
2. **Last Name**: 성
3. **Organization Name**: 조직명 [선택사항]
4. **Phone Number**: 전화번호 [선택사항]

<Step number={3} title="청구 정보 설정 [선택사항]">

크레딧이 소진된 후 API를 계속 사용하려면 청구 정보를 설정해야 합니다.

### 결제 수단 추가

1. 계정 설정으로 이동
2. **Billing** 또는 **Payment** 탭
3. **Add payment method** 클릭
4. 신용카드 또는 직불카드 정보 입력

### 사용 한도 설정 [권장]

비용을 제어하기 위해 한도를 설정합니다:

1. **Billing** 탭
2. **Usage limits** 섹션
3. **Soft limit**: 경고를 받는 금액 [예: $10]
4. **Hard limit**: API 중단되는 금액 [예: $20]

<Callout type="warning">
⚠️ **비용 주의사항**

- API 호출 기반 종량제
- 모델마다 다른 가격
- 한도 설정으로 예상치 못한 비용 방지
- 정기적으로 사용량 확인

가격 예시:
- gpt-3.5-turbo: 매우 저렴
- gpt-4: 더 비싸지만 성능 우수
- DALL-E: 이미지당 청구
</Callout>

<Step number={4} title="API 키 생성 및 확보">

### API 키 생성

1. [OpenAI API 대시보드](https://platform.openai.com)에 로그인합니다.
2. 왼쪽 메뉴에서 **API keys** 또는 **Billing and usage** 클릭
3. **Create new secret key** 또는 **+Create new secret key** 버튼 클릭

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/v1/openai/openai-account-setup-step4-ko.png"
  alt="API 키 생성"
  caption="OpenAI 대시보드에서 API 키 생성"
/>

### API 키 복사 및 저장

1. 생성된 API 키가 표시됩니다.
2. **Copy** 버튼을 클릭하여 복사합니다.
3. 안전한 곳에 저장합니다 [예: 비밀번호 관리자]

<Callout type="warning">
⚠️ **API 키 보안 주의사항**

- API 키를 절대 공개하지 마세요
- 코드에 직접 작성하지 마세요
- .env 파일에 저장하세요
- GitHub에 커밋하면 안 됨
- 유출된 경우 즉시 재생성하세요

잘못된 예:
```
const apiKey = 'sk-xxxxx'; // 절대 금지!
```

올바른 예:
```
const apiKey = process.env.OPENAI_API_KEY;
```
</Callout>

## 문제 해결

### "Authentication failed"

- API 키가 정확한지 확인
- API 키가 복사되었는지 확인
- API 키에 "sk-" 접두사 있는지 확인

### "Rate limit exceeded"

- API 호출 빈도 제한
- 사용 한도 확인
- 잠시 후 다시 시도

### 키가 작동하지 않음

- API 키 재생성
- 새로운 .env 파일 생성
- 환경 변수 다시 로드
