# Firebase Project Setup - Get Started for Free

Firebase is a **backend service platform** provided by Google that allows app developers to use databases, authentication, storage, and more without managing their own servers. Follow this guide to create a Firebase project and start with the free Spark Plan in just 15 minutes.

<Step number={2} title="Create a New Project">

1. Click the **"Create a project"** button
2. Enter your **project name** (e.g., "My App", "Blog Project", etc.)
3. Click **Next**
4. Configure Google Analytics (optional)

   - Enable this if you want Firebase analytics
5. Click **Create project**

<Step number={5} title="Add App and Get Configuration Code">

Now you can add a web, iOS, or Android app and get the Firebase SDK configuration code.

1. Go to the **Project Overview** page in Firebase Console
2. Click the **"Add app"** button
3. Select **Web platform** (or iOS/Android)
4. Enter your **app nickname** (e.g., "My Web App")
5. Choose **Firebase Hosting setup** (optional)
6. Click **Register app**
7. Copy the **Firebase SDK configuration** code

## Troubleshooting

<DevTip>
**"Project creation failed"**

If you see this error:
1. Verify your Google account is in good standing
2. Try a different browser or incognito mode
3. Clear browser cache and try again
4. Check the Firebase status page (https://status.firebase.google.com) for any outages
</DevTip>

<Callout type="warning">
‚ö†Ô∏è **Project ID Considerations**

- Project ID cannot be changed after creation
- It's included in your Firebase app configuration code, so handle it securely
- API keys are safe to make public, but always secure your database with proper rules
</Callout>

## Firebase Services Overview

**Realtime Database vs Firestore - Which one to choose?**

| Feature | Realtime Database | Firestore |
|---------|-------------------|-----------|
| Data Structure | JSON | Collections + Documents |
| Querying | Limited | Powerful queries |
| Scalability | Lower | Higher |
| Recommended For | Simple apps | Medium to large apps |

Spark Plan Limits:
- Realtime Database: 100 concurrent connections, 100 operations/hour
- Firestore: 50,000 reads/day, 20,000 writes/day
- Storage: 5GB limit
