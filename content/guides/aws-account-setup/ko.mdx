# AWS 계정 만들기 - 12개월 무료 체험 시작하기

Amazon Web Services(AWS)는 **전 세계 1위 클라우드 플랫폼**으로, **12개월 무료 체험**을 제공합니다. 이 가이드를 따라 AWS 계정을 생성하고 10분 안에 무료 체험을 시작해보세요!

## AWS Free Tier로 무엇을 할 수 있나요?

<Callout type="info">
☁️ **AWS 12개월 무료 체험 혜택**

- **EC2**: 월 750시간 t2.micro 인스턴스
- **S3**: 5GB 표준 스토리지
- **RDS**: 월 750시간 db.t2.micro 데이터베이스
- **Lambda**: 월 100만 건 요청
- **DynamoDB**: 25GB 스토리지
- **CloudFront**: 50GB 데이터 전송

**영구 무료**:
- Lambda 월 100만 건
- DynamoDB 25GB
- SNS 100만 건 발행
</Callout>

1. [AWS 홈페이지](https://aws.amazon.com/ko/)에 접속합니다.
2. 우측 상단 **AWS 계정 생성** 버튼을 클릭합니다.

### 루트 사용자 이메일 주소 입력

1. **루트 사용자 이메일 주소**: 이메일 입력
2. **AWS 계정 이름**: 계정 이름 입력 (예: "내 AWS 계정")
3. **이메일 주소 확인**: 동일한 이메일 재입력
4. **계속(Continue)** 클릭

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/aws-account-step1b-ko.png"
  alt="이메일 및 계정 이름 입력"

<Callout type="info">
**루트 사용자란?**

루트 사용자는 AWS 계정의 **최고 권한자**입니다. 모든 AWS 서비스와 리소스에 접근할 수 있으므로, 보안을 위해 나중에 IAM 사용자를 만들어 사용하는 것이 좋습니다.
</Callout>

### 이메일 인증

1. 입력한 이메일로 **인증 코드** 발송
2. 이메일에서 인증 코드 확인
3. **인증 코드 입력** 필드에 코드 입력
4. **확인(Verify)** 클릭

<Step number={2} title="비밀번호 및 연락처 정보 입력">

### 루트 사용자 비밀번호 설정

1. **비밀번호** 입력:
   - 최소 8자 이상
   - 대문자, 소문자, 숫자, 특수문자 포함 권장
2. **비밀번호 확인**: 동일한 비밀번호 재입력
3. **계속** 클릭

<Callout type="tip">
💡 **강력한 비밀번호 예시**: `MyAws2024!Secure`
</Callout>

### 연락처 정보 입력

1. **계정 유형** 선택:
   - **개인(Personal)**: 개인 학습/프로젝트용
   - **비즈니스(Business)**: 회사/조직용

2. **전체 이름**: 본인 이름 입력
3. **전화번호**: 휴대폰 번호 입력 (예: 010-1234-5678)
4. **국가/지역**: 대한민국 선택
5. **주소**: 정확한 주소 입력
6. **우편번호**: 우편번호 입력

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/aws-account-step2-ko.png"
  alt="연락처 정보 입력"
  caption="정확한 연락처 정보를 입력하세요"
/>

7. **AWS 고객 계약 동의** 체크박스 선택
8. **계속** 클릭

</Step>

<Step number={3} title="결제 정보 입력">

### 신용카드 정보 등록

1. **카드 번호**: 16자리 카드 번호 입력
2. **만료일**: 카드 만료일 (MM/YY)
3. **카드 소유자 이름**: 카드에 표시된 이름
4. **청구지 주소**: Step 2에서 입력한 주소와 동일

<Screenshot
  src="https://res.cloudinary.com/demo/image/upload/aws-account-step3-ko.png"

  caption="신용카드 정보를 입력하세요"
/>

5. **확인 및 계속(Verify and Continue)** 클릭

<Callout type="info">
**1달러 임시 승인**

본인 확인을 위해 1달러가 일시적으로 청구됩니다. 이는 **2-3일 내 자동 환불**되며, 실제 청구가 아닙니다.
</Callout>

### 본인 확인 (SMS 또는 전화)

1. **인증 방법** 선택:
   - **문자 메시지(SMS)** (권장)
   - **음성 통화**

2. 휴대폰 번호 확인
3. **SMS 전송(Send SMS)** 클릭
4. 받은 **4자리 인증 코드** 입력
5. **코드 확인** 클릭

무료 체험 중 다음 사항을 주의하세요:

1. **EC2 인스턴스**: t2.micro만 무료 (월 750시간)

## 자주 묻는 질문

<Callout type="info">
**Q. 무료 체험 후 자동으로 과금되나요?**

A. 네, 12개월 후 또는 무료 한도 초과 시 자동으로 과금됩니다.
- 해결: Billing Alerts 설정으로 예산 초과 방지
- 사용하지 않는 리소스는 즉시 삭제
</Callout>

<Callout type="info">
**Q. 신용카드 없이 가입할 수 있나요?**

A. 아니요. AWS는 **신용카드 등록이 필수**입니다. 체크카드도 가능하지만, 일부 은행 카드는 등록이 안 될 수 있습니다.
</Callout>

<Callout type="info">
**Q. 계정 활성화가 안 돼요**

A. 다음을 확인하세요:
- 신용카드 정보가 정확한지 확인
- SMS 인증이 완료되었는지 확인
- 24시간 대기 (보통 몇 분 내 완료)
- AWS Support에 문의
</Callout>

<Callout type="info">
**Q. 여러 개의 AWS 계정을 만들 수 있나요?**

A. 네, 이메일 주소만 다르면 여러 계정을 만들 수 있습니다. 하지만 각 계정마다 신용카드 등록이 필요하며, 무료 체험은 계정당 1회만 가능합니다.
</Callout>

## 문제 해결

### "Invalid payment method" 오류

- 카드 정보가 정확한지 확인
- 해외 결제가 가능한 카드인지 확인
- 다른 카드로 시도
- 은행에 문의하여 해외 결제 차단 해제

### SMS 인증 번호가 오지 않아요

- 휴대폰 번호가 정확한지 확인
- 스팸 차단 설정 확인
- 음성 통화 인증으로 전환
- 5분 후 재시도

### 계정이 비활성화되었어요

- AWS Support에 문의
- 결제 정보 재확인
- 본인 확인 서류 제출 필요할 수 있음
