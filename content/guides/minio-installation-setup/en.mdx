  -e MINIO_ROOT_PASSWORD=minioadmin \\

<CopyBlock
  code="from minio import Minio
import os

# Connect to MinIO
client = Minio(
  'localhost:9000',
  access_key='minioadmin',
  secret_key='minioadmin',
  secure=False  # False for local, True for production
)

# Create bucket (if not exists)
if not client.bucket_exists('test-bucket'):
  client.make_bucket('test-bucket')

# Upload file
client.fput_object(
  'test-bucket',
  'my-file.txt',
  'path/to/local/file.txt'
)

print('File uploaded successfully!')

# List buckets
buckets = client.list_buckets()
for bucket in buckets:
  print(f'Bucket: {bucket.name}')"

## Common Issues

### Port Already in Use

```
Error: listen tcp :9000: bind: address already in use
```

Solutions:
- Change port: `-p 9010:9000`
- Kill process using port: `lsof -i :9000`
- Use different port range: `-p 9000-9002:9000-9002`

### Docker Connection Issues

```
Cannot connect to localhost:9000
```

Solutions:
- Check Docker is running
- Verify port forwarding: `docker ps`
- Use proper Docker IP address
- For Windows/Mac, use `host.docker.internal`
